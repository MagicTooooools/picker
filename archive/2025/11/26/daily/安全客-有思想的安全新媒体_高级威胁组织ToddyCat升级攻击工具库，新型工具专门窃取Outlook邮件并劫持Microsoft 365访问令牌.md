### ToddyCat зҡ„ж”»еҮ»жј”иҝӣдёҺе·Ҙе…·й“ҫ

ToddyCat иҮӘ 2020 е№ҙиө·жҙ»и·ғпјҢжӣҫдҪҝз”ЁВ **Samurai**В е’ҢВ **TomBerBil**В зӯүе·Ҙе…·й’ҲеҜ№ж¬§дәҡең°еҢәеӨҡе®¶з»„з»ҮпјҢд»Ҙз»ҙжҢҒи®ҝй—®жқғе№¶зӘғеҸ–и°·жӯҢ ChromeгҖҒеҫ®иҪҜ Edge зӯүжөҸи§ҲеҷЁзҡ„ cookies е’ҢеҮӯжҚ®гҖӮ

2024 е№ҙ 4 жңҲпјҢиҜҘз»„з»Үиў«жҢҮеҲ©з”Ё ESET е‘Ҫд»ӨиЎҢжү«жҸҸеҷЁжјҸжҙһпјҲCVE-2024-11859пјҢCVSS иҜ„еҲҶ 6.8пјүжҠ•ж”ҫжңӘе…¬ејҖзҡ„жҒ¶ж„ҸиҪҜд»¶В **TCESB**гҖӮ

еҚЎе·ҙж–ҜеҹәеңЁ 2024 е№ҙ 5-6 жңҲзҡ„ж”»еҮ»дёӯеҸ‘зҺ°дәҶ TomBerBil зҡ„В **PowerShell еҸҳдҪ“**пјҲжӯӨеүҚдёә C++ е’Ң C# зүҲжң¬пјүпјҢж–°еўһдәҶд»Һ Mozilla Firefox жҸҗеҸ–ж•°жҚ®зҡ„еҠҹиғҪгҖӮиҜҘзүҲжң¬зҡ„жҳҫи‘—зү№зӮ№жҳҜВ **д»Ҙзү№жқғз”ЁжҲ·иә«д»ҪеңЁеҹҹжҺ§еҲ¶еҷЁдёҠиҝҗиЎҢ**пјҢе№¶йҖҡиҝҮ SMB еҚҸи®®и®ҝй—®е…ұдә«зҪ‘з»ңиө„жәҗдёӯзҡ„жөҸи§ҲеҷЁж–Үд»¶гҖӮ

![]()

жҒ¶ж„ҸиҪҜд»¶йҖҡиҝҮи®ЎеҲ’д»»еҠЎжү§иЎҢ PowerShell е‘Ҫд»ӨеҗҜеҠЁпјҢе…·дҪ“иҖҢиЁҖпјҢе®ғйҖҡиҝҮ SMB жҗңзҙўиҝңзЁӢдё»жңәдёӯзҡ„жөҸи§ҲеҷЁеҺҶеҸІи®°еҪ•гҖҒcookies е’Ңдҝқеӯҳзҡ„еҮӯжҚ®гҖӮе°Ҫз®ЎеҢ…еҗ«дҝЎжҒҜзҡ„еӨҚеҲ¶ж–Үд»¶дҪҝз”Ё Windows ж•°жҚ®дҝқжҠӨ APIпјҲDPAPIпјүеҠ еҜҶпјҢдҪҶ TomBerBil иғҪеӨҹжҚ•иҺ·и§ЈеҜҶжүҖйңҖзҡ„еҠ еҜҶеҜҶй’ҘгҖӮ

з ”з©¶дәәе‘ҳи§ЈйҮҠпјҡвҖңж—§зүҲ TomBerBil еңЁдё»жңәдёҠиҝҗиЎҢе№¶еӨҚеҲ¶з”ЁжҲ·д»ӨзүҢпјҢйҖҡиҝҮ DPAPI и§ЈеҜҶз”ЁжҲ·еҪ“еүҚдјҡиҜқдёӯзҡ„дё»еҜҶй’ҘпјҢиҝӣиҖҢи§ЈеҜҶж–Үд»¶пјӣиҖҢж–°зүҲжңҚеҠЎеҷЁзүҲжң¬еҲҷеӨҚеҲ¶ DPAPI дҪҝз”Ёзҡ„з”ЁжҲ·еҠ еҜҶеҜҶй’Ҙж–Үд»¶пјҢз»“еҗҲз”ЁжҲ· SID е’ҢеҜҶз ҒпјҢж”»еҮ»иҖ…еҸҜеңЁжң¬ең°и§ЈеҜҶжүҖжңүеӨҚеҲ¶ж–Үд»¶гҖӮвҖқ

### й’ҲеҜ№йӮ®д»¶ж•°жҚ®зҡ„дё“йЎ№жҠҖжңҜ

ToddyCat иҝҳдҪҝз”Ё TCSectorCopyпјҲвҖңxCopy.exe вҖқпјүи®ҝй—®жң¬ең° Microsoft Outlook еӯҳеӮЁзҡ„В **OST ж–Үд»¶**пјҲи„ұжңәеӯҳеӮЁиЎЁпјүпјҢз»•иҝҮеә”з”ЁиҝҗиЎҢж—¶еҜ№ж–Үд»¶зҡ„и®ҝй—®йҷҗеҲ¶гҖӮ

TCSectorCopy д»Ҙ C++ зј–еҶҷпјҢжҺҘеҸ—еҫ…еӨҚеҲ¶ж–Үд»¶пјҲжӯӨеӨ„дёә OST ж–Үд»¶пјүдҪңдёәиҫ“е…ҘпјҢе°ҶзЈҒзӣҳдҪңдёәеҸӘиҜ»и®ҫеӨҮжү“ејҖе№¶жҢүжүҮеҢәйЎәеәҸеӨҚеҲ¶ж–Үд»¶еҶ…е®№гҖӮOST ж–Үд»¶иў«еҶҷе…Ҙж”»еҮ»иҖ…жҢҮе®ҡи·Ҝеҫ„еҗҺпјҢеҶҚйҖҡиҝҮејҖжәҗе·Ҙе…·В **XstReader**В жҸҗеҸ–йӮ®д»¶еҶ…е®№гҖӮ

### дә‘зҺҜеўғд»ӨзүҢзӘғеҸ–дёҺз»•иҝҮжүӢж®ө

й’ҲеҜ№дҪҝз”Ё Microsoft 365 дә‘жңҚеҠЎзҡ„еҸ—е®іиҖ…пјҢToddyCat е°қиҜ•зӣҙжҺҘд»ҺеҶ…еӯҳиҺ·еҸ–и®ҝй—®д»ӨзүҢпјҡйҖҡиҝҮејҖжәҗ C# е·Ҙе…·В **SharpTokenFinder**В жһҡдёҫ Microsoft 365 еә”з”ЁзЁӢеәҸпјҢеҜ»жүҫжҳҺж–Үиә«д»ҪйӘҢиҜҒ JSON Web д»ӨзүҢпјҲJWTпјүгҖӮ

дҪҶеңЁиҮіе°‘дёҖиө·дәӢд»¶дёӯпјҢзі»з»ҹе®үе…ЁиҪҜд»¶йҳ»жӯўдәҶ SharpTokenFinder иҪ¬еӮЁВ `Outlook.exe`В иҝӣзЁӢзҡ„е°қиҜ•гҖӮдёәз»•иҝҮйҷҗеҲ¶пјҢж”»еҮ»иҖ…дҪҝз”Ё Sysinternals е·Ҙе…·еҢ…дёӯзҡ„В **ProcDump**пјҢйҖҡиҝҮзү№е®ҡеҸӮж•°еҜ№ Outlook иҝӣзЁӢиҝӣиЎҢеҶ…еӯҳиҪ¬еӮЁгҖӮ

еҚЎе·ҙж–ҜеҹәжҢҮеҮәпјҡвҖңToddyCat APT з»„з»ҮжҢҒз»ӯеҸ‘еұ•жҠҖжңҜпјҢеҜ»жүҫиғҪйҡҗи—Ҹжҙ»еҠЁзҡ„ж–№жі•пјҢд»ҘеңЁеҸ—жҚҹеҹәзЎҖи®ҫж–ҪеҶ…иҺ·еҸ–дјҒдёҡйҖҡдҝЎи®ҝй—®жқғгҖӮвҖқ
