## Watchy：为开发者打造的开源电子墨水智能手表

Watchy 是一款完全开源的 E-Ink 智能手表，由 SQFMI 团队开发，基于 ESP32 平台构建，支持蓝牙低功耗（BLE）、Arduino 编程和电子墨水屏显示。它不仅提供完整的硬件设计与软件代码，也为用户自定义表盘和功能开发提供了广阔空间。

![Watchy开源电子墨水智能手表：基于ESP32的可定制穿戴设备](https://cdn.skyimg.net/up/2025/11/26/aad6ae61.webp)

### 核心特点

* **开源架构**：软件与硬件设计全部开源，代码托管于 GitHub，允许任意修改与使用。
* **ESP32 内核**：搭载功能强大的 ESP32 微控制器，支持 Wi-Fi 与 BLE 双模连接。
* **电子墨水显示**：使用低功耗 E-Ink 显示屏，确保长时间续航与清晰显示效果。
* **丰富示例**：内置多种表盘模板，开发者可快速上手、自由拓展。
* **Arduino 支持**：完全兼容 Arduino 开发环境，适合创客、学生及工程师使用。

### 入门与资源

用户可通过官方文档（[Watchy文档站](https://watchy.sqfmi.com/docs/getting-started)）快速开始项目，包括手表组装、表盘开发、固件上传等流程。所有源代码与示例均可在 GitHub 仓库（[GitHub项目页](https://github.com/sqfmi/Watchy)）中找到。

### 购买与配件

这些渠道均提供 Watchy 主体及相关配件模块，便于用户个性化组装与功能拓展。

### 开发者社区与协议

项目鼓励用户参与开发与分享，开发者可加入 Watchy Discord 社区 与全球用户互动交流。Watchy 采用 MIT 开源协议，支持商用与二次开发。

## Watchy 适合谁？

* 对物联网感兴趣的开发者与创客
* 想构建个人化可穿戴设备的技术爱好者
* 寻找低功耗可编程手表平台的工程师
* 对 Arduino 和 ESP32 项目有开发经验的学生或研究人员

这不仅是一块表，更是一块面向未来的开发板。
