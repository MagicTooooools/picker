## 抖音记录 使用方法

2025-11-27 / 1 min read
[#其它](https://raz1ner.com/tag/other/)

自动获取抖音关注数、粉丝数、好友数写入表格。

## 使用方法

1. 新建一个电子表格，然后创建两个分页，分别是「账号记录」和「账号」。
   * 「账号记录」分页的表头可以按照下面截图的模版设置。
   * 「账号」分页的 A 列放入抖音账号主页的链接。

![](https://raz1ner.com/post-images/1764288726655.png)

2. 在表格点击「扩展程序」→「Apps 脚本」，放入 [code.gs](https://github.com/dev-coco/TikTok-Activity-Tracker/blob/main/code.gs) 代码。
3. 部署 API 脚本。点击右上角的「部署」→「新建部署」。
   ![](https://raz1ner.com/post-images/1764289159736.png)

类型选择「Web应用」。

![](https://raz1ner.com/post-images/1764289230533.png)

Web 应用的执行身份选择「我」，有访问权限的人员选择「任何人」，点击部署。

![](https://raz1ner.com/post-images/1764289235722.png)

成功授予访问权限后，会获得一个 API 脚本链接。

![](https://raz1ner.com/post-images/1764289355659.png)

4. 插件下载：[Chrome 插件合集](/Extension/?active=%E6%8A%96%E9%9F%B3%E8%AE%B0%E5%BD%95)

安装完成后，点击插件，放入部署好的 API 脚本链接。

![](https://raz1ner.com/post-images/1764289854647.png)

当浏览器打开新的窗口的时候，就会自动在后台获取每一个抖音主页的数据，然后写入表格。
