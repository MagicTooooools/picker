# Awesome Nano Banana Pro

A curated, open-source collection of high-quality prompts and examples for the Nano Banana Pro (Nano Banana 2) image model.

Author:
ZeroLu

Since:
2025-11-10

[Visit Website](https://blog.google/products/gemini/prompting-tips-nano-banana-pro/) [GitHub](https://github.com/ZeroLu/awesome-nanobanana-pro)

## Detailed Introduction

Awesome Nano Banana Pro is an open-source curated list maintained by ZeroLu that aggregates high-quality prompts, examples, and usage notes for the Nano Banana Pro (Nano Banana 2) image model. The repository organizes effective prompts sourced from X (Twitter), WeChat, Replicate and other communities into thematic sections, enabling practitioners and researchers to learn prompt engineering techniques, reproduce creative effects, and quickly generate high-fidelity images.

## Main Features

* Curated prompts: ready-to-use high-quality prompts covering photorealism, creative experiments, education, e-commerce, and more.
* Well-organized: thematic categories make it easy to find and contribute new examples.
* Open license: published under MIT for easy community reuse and iteration.
* Community-sourced: examples collected from social platforms with source attribution to ensure reproducibility.

## Use Cases

* Prompt engineering study and reference to speed up mastering Nano Banana Pro for different scenarios.
* Creative asset library for advertising, short-video thumbnails, e-commerce imagery, and educational infographics.
* Teaching and research: classroom examples, model evaluations, and comparative experiments.

## Technical Features

* Maintained as Markdown on GitHub for collaborative updates via pull requests.
* Focuses on image-generation prompts with attention to composition, lighting, material, and style control to boost model outputs.
* README includes quickstart instructions and reference resources; pair with official prompting guide for deeper optimization.
