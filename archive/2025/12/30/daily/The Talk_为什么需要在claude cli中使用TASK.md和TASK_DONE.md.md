# 为什么需要在claude cli中使用TASK.md和TASK\_DONE.md

LLM杂谈

TASK.md 是一种轻量级 AI Agent 任务管理模式，专为解决 Claude Code 等长时间运行代理的会话间状态丢失问题而设计。基于 Anthropic 官方双代理架构理念，该模式通过简单的 Markdown 文件实现任务队列、上下文传递和进度追踪的持久化。相比纯提示词记忆或复杂数据库方案，TASK.md 具有 Git 原生集成、人类可读、零成本部署的优势。文章详解了最佳实践模板，包括任务分解、验收标准、技术决策记录等结构，帮助开发团队实现透明协作和无缝任务交接，是提升 AI 辅助编程效率的实用工程方法。
