# Beyond good ol’ Run key, Part 152

Posted on [2025-12-31](https://www.hexacorn.com/blog/2025/12/31/beyond-good-ol-run-key-part-152/ "11:08 pm")  by  [adam](https://www.hexacorn.com/blog/author/adam/ "View all posts by adam")

Forgot to post this one before [153](https://www.hexacorn.com/blog/2025/10/25/beyond-good-ol-run-key-part-153/)…

As this is often the case in this series, this persistence mechanism is a [documented feature of Microsoft Windows](https://learn.microsoft.com/en-us/troubleshoot/windows/win32/xa-transaction-require-registry-entry).

The following Registry entry:

```
HKLM\Software\Microsoft\MSDTC\XADLL
```

can store a list of DLL files that can be loaded by the OS.
