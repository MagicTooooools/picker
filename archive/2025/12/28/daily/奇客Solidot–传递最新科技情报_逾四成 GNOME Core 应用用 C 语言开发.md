## 逾四成 GNOME Core 应用用 C 语言开发

[![GNOME](https://icon.solidot.org/images/topics/topicgnome.png?123)](/search?tid=74 "GNOME")

[Edwards](/~Edwards) (42866)发表于 2025年12月28日 23时19分 星期日 [新浪微博分享](//service.weibo.com/share/share.php?url=//www.solidot.org/story?sid=83164&appkey=1370085986&title=%E9%80%BE%E5%9B%9B%E6%88%90%20GNOME%20Core%20%E5%BA%94%E7%94%A8%E7%94%A8%20C%20%E8%AF%AD%E8%A8%80%E5%BC%80%E5%8F%91 "新浪微博分享")
![](https://icon.solidot.org/images/a7c7.png)

**来自梦海**

GNOME 开发者分享了该项目的代码和语言统计数据：截至 2025 年底，总代码行数 6,692,516 行，其中应用 1,611,526 行。44.8% 的 GNOME Core 应用用 C 语言开发，之后是 20.7% 的 Vala，13.8% 的 JavaScript，Rust 语言开发的应用占 10.3%，Python 占 6.9%，C++ 占 3.45%。GNOME Circle Apps 中最流行的语言是 Rust，占比高达 41.7%，C 仅占 5.56%，Python 29.2%，Vala 12.5%，JavaScript 9.72%。C 语言在 GNOME 组件/库中占主导位置，占比 76%，其次是 Rust 的 10.3% 和 Vala 的 3.77%。
https://www.phoronix.com/news/GNOME-2025-Code-Stats
https://blogs.gnome.org/sophieh/2025/12/27/gnome-in-2025-some-numbers/
