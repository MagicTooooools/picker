# Sony Float Run Open-Ear Wireless Headphones

12-20-202512-20-2025 [essay](/essay/) 6 minutes read (About 834 words)  visits

前阵子，我使用两年多的 [Sony Float Run Open-Ear Wireless Headphones](https://electronics.sony.com/audio/headphones/all-headphones/p/wioe610-b?srsltid=AfmBOorwG1jaWs5e3jdgg0gsrVZmeqjzgxe8ppeECCyniv6pCxEiwSTk) 左边没有声音了，正式寿终正寝。今天我在这里说说对它的使用感受。

![Sony Float Run](https://raw.githubusercontent.com/leimao-servant/Blog-Images-Backup/main/essay/2025-12-20-Sony-Float-Run-Open-Ear-Wireless-Headphones/FloatRun_Intro_01_M.webp?raw=true "Sony Float Run")

我觉得 2023 年发售的 Sony Float Run Open-Ear Wireless Headphones 这款耳机是世界上第一款真正意义上的外放耳机，专为跑步运动设计。在那之前，我用过 Apple AirPods 和 [Sony Walkman NW-WS413](/essay/Sony-Walkman-NW-WS413/) 作为跑步时候听歌的耳机，但是感觉都不是很好。Apple AirPods 经常会因为汗水，跑着跑着就掉了。Sony Walkman NW-WS413 的入耳式设计太硬，会让耳道不舒服。所以我在得知世界上居然有外放耳机这种东西的时候，立刻就买了一副来试试。

![Sony Float Run](https://raw.githubusercontent.com/leimao-servant/Blog-Images-Backup/main/essay/2025-12-20-Sony-Float-Run-Open-Ear-Wireless-Headphones/FloatRun_Intro_02_M.webp?raw=true "Sony Float Run")

就佩戴舒适性而言，这款耳机确实极好的。它的重量非常轻，佩戴上也根本不用担心会掉下来。因为是外放式设计，所以无论佩戴多久，对耳朵也没有压力。再加上耳机防汗水，简直完美适配跑步这项运动。需要指出的是，因为这副耳机是挂在耳朵上的，如果同时佩戴眼镜的话会有些许冲突，但总体不影响佩戴耳机时候的感受。

![Sony Float Run](https://raw.githubusercontent.com/leimao-servant/Blog-Images-Backup/main/essay/2025-12-20-Sony-Float-Run-Open-Ear-Wireless-Headphones/FloatRun_Intro_03_M.webp?raw=true "Sony Float Run")

比较遗憾的是，在使用这款耳机大概一年半之后，耳机就出现了开胶的情况。虽然似乎不影响功能性，但是考虑到这个是 Sony 制造的耳机，这个就很难接受了。在那之后又过了半年，左耳就彻底没有声音了，也就正式放弃了这款耳机。

![Sony Float Run](https://raw.githubusercontent.com/leimao-servant/Blog-Images-Backup/main/essay/2025-12-20-Sony-Float-Run-Open-Ear-Wireless-Headphones/FloatRun_Intro_04_M.webp?raw=true "Sony Float Run")

比较可惜的是，Sony 推出这款耳机之后，似乎并没有在市场上掀起很大的波澜。自发布至今，近三年过去，Sony 也没有推出第二代或者类似的外放耳机产品。倒是 Bose 和 Shokz 等品牌，抄了 Sony 的作业，从 2024 年开始也推出了类似的外放式运动耳机的产品。

![Sony Float Run](https://raw.githubusercontent.com/leimao-servant/Blog-Images-Backup/main/essay/2025-12-20-Sony-Float-Run-Open-Ear-Wireless-Headphones/FloatRun_Intro_05_M.webp?raw=true "Sony Float Run")

电子产品买新不买旧。很可惜 Sony 并没有推出新的外放式运动耳机产品，我只好购买其他品牌的外放式运动耳机产品来陪伴我跑步了。
