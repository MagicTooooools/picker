## 打造你的专属 Netflix：冬瓜TV MAX 全能聚合播放器

冬瓜TV MAX 是基于 Node.js 与 Vue 3 构建的开源流媒体聚合播放器，支持多源数据整合、TMDb 海报与评分解析、智能搜索、自动线路测速切换等功能，为影视爱好者打造沉浸式的播放体验。相比原版项目，冬瓜TV MAX 实现了深度 UI 优化与使用体验革新。

[![冬瓜TV MAX：开源私有流媒体聚合平台，TMDb 智能刮削打造你的私人 Netflix](https://cdn.skyimg.net/up/2025/12/20/f9d70edc.webp)](https://www.ahhhhfs.com/wp-content/uploads/2025/12/%E5%86%AC%E7%93%9CTV-MAX%EF%BC%9A%E5%BC%80%E6%BA%90%E7%A7%81%E6%9C%89%E6%B5%81%E5%AA%92%E4%BD%93%E8%81%9A%E5%90%88%E5%B9%B3%E5%8F%B0%EF%BC%8CTMDb-%E6%99%BA%E8%83%BD%E5%88%AE%E5%89%8A%E6%89%93%E9%80%A0%E4%BD%A0%E7%9A%84%E7%A7%81%E4%BA%BA-Netflix-02.jpg)

### 🌟 智能聚合 + 极速播放

* **多源数据整合**：接入 TMDb 影视元数据与 48+ Maccms 第三方资源站 API，支持实时测速与智能筛选，避免卡顿源。
* **实时搜索反馈**：采用 SSE 流式技术，搜索结果边搜边显，无需等待所有数据返回。
* **线路故障自动切换**：播放失败自动切换可用线路，无需手动处理。

### 💻 极简部署 + 跨平台支持

* **Vercel 一键部署**：开箱即用，支持国际与国内访问优化。
* **Docker 本地部署**：适合搭建在私人服务器，支持 sqlite/json 缓存选择。
* **Android TV 与移动端支持**：专属 APK 与 PWA 封装，体验接近原生应用。

### 🔒 安全稳定 + 用户体验至上

* **访问密码保护**：可设置全局访问密码，提升站点私密性。
* **本地静态资源优先**：提高国内加载速度，避免 CDN 劫持。
* **远程配置与多端同步**：支持远程加载配置文件，便于多站点统一管理。

### 🎨 全新 UI 设计，沉浸式体验

| 功能模块 | 原版体验 | MAX 版本升级 |
| --- | --- | --- |
| 首页视觉 | 普通列表展示 | Netflix 风格轮播 + 动态背景 + 排名特效 |
| 搜索体验 | 静态加载 | 实时结果跳动呈现 + 动态聚合 |
| 分类系统 | 搜索跳转 | 独立分类榜单（如历史、冒险、综艺） |
| 播放页面 | 常规播放页 | 剧集网格 + 自动测速 + 故障转移 + 投屏支持 |
| 导航栏与交互 | 固定顶部 | 滚动变色 + 胶囊悬浮搜索框 + 回到顶部按钮 |

### ⚙️ 核心技术栈

* **前端**：Vue 3, Bootstrap 5, FontAwesome, DPlayer, HLS.js
* **后端**：Node.js, Express, Axios
* **数据源**：TMDb API v3, Maccms API 聚合
* **缓存系统**：SQLite（推荐）、JSON 文件、本地内存
* **部署方式**：Docker、Vercel、PM2、宝塔面板
* **国内优化**：Cloudflare Workers、反代地址支持
