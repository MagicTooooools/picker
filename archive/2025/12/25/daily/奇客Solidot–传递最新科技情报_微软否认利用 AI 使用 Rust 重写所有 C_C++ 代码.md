## 微软否认利用 AI 使用 Rust 重写所有 C/C++ 代码

[![微软](https://icon.solidot.org/images/topics/topicMicrosoft.png?123)](/search?tid=150 "微软")

[Edwards](/~Edwards) (42866)发表于 2025年12月25日 14时28分 星期四 [新浪微博分享](//service.weibo.com/share/share.php?url=//www.solidot.org/story?sid=83146&appkey=1370085986&title=%E5%BE%AE%E8%BD%AF%E5%90%A6%E8%AE%A4%E5%88%A9%E7%94%A8%20AI%20%E4%BD%BF%E7%94%A8%20Rust%20%E9%87%8D%E5%86%99%E6%89%80%E6%9C%89%20C%2FC%2B%2B%20%E4%BB%A3%E7%A0%81 "新浪微博分享")
![](https://icon.solidot.org/images/a7c7.png)

**来自电动新世纪**

微软杰出工程师 Galen Hunt 在 LinkedIn 上畅谈要在 AI 和算法的帮助下，到 2030 年用 Rust 语言重写所有 C 和 C++ 代码，目标是一名工程师一个月一百万行代码。这番话引发了一片哗然，以至于微软进行澄清，而 Galen Hunt 则修改了他的帖子。他在帖子里大量使用“我们（We）”这个词，因此在外界看来他是代表公司发言。微软高管 Frank X. Shaw 澄清，公司没有计划使用 AI 重写 Windows 11。而 Galen Hunt 也在帖子里澄清：微软没有采用 AI 使用 Rust 重写 Windows 11，表示读者过度解读了。
https://www.windowslatest.com/2025/12/24/microsoft-denies-rewriting-windows-11-using-ai-after-an-employees-one-engineer-one-month-one-million-code-post-on-linkedin-causes-outrage/
