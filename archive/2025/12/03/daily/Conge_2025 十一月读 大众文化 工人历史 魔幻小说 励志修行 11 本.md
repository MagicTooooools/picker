### 01. [文化理论与大众文化导论](https://neodb.social/book/3niPeoFtIv63A9GYdbQ9EK)

|  |  |  |
| --- | --- | --- |
| 2025-11-16 读过 | 哲学，文化理论，文化研究，精神分析 | 豆瓣评分 9.3 |

我给它【7 分】：凡有所相，皆是虚妄。

什么是文化？当我看一部电影，听一首歌，观一场表演，读一本书，吃一顿大餐，跳一段舞，我只是在做一件事，不会意识到这是文化活动。与其相区别的，还有政治、经济等活动。文化神奇的之处，便是通过对其他活动的描写记录，将所有活动“文化”，形成文本。于是，一切可言说的可实践的，都可被装进文化的篮子，作为文本的和实践的文化存在。“文”即纹，表面、也就是象征。这是我对文化的理解，不是本书的定义。文化包括虚的意识形态和实的物质、活动、关系。..

什么是大众文化？源于工人阶级的生产活动之外的自发的消磨时间的方式，被权贵学者视为群氓，被出身于工人阶级的精英学者者视为本真，很快被商业视为赚钱工具。..。从产生来源看，文化工业出产的被斥为群氓，对于工人阶级自发产生的文化则被当作本真。从消费端看，有的学者视大众为被动接受统治阶级商业文化洗脑的群氓，有的认为消费过程的拣选就是大众的创造性力量。借由消费者主动的选择，文化产品才能流行，成为大众文化，不然就是地下的，小众的。

文化理论，则是对大众文化发表各种观点的学者的看法。文化，文明与文化传统派看不上大众文化，认为它的生产者和受众应当接受教育，去矫正粗鄙的审美；文化分析派则对大众文化有褒有贬有拣选，仍倡导对其中不好的部分用教育去粗取精。后现代主义则对学者高高在上的指手画脚嗤之以鼻，主张研究是什么，而不发表好坏的见解：偏好与事实无关。

什么是意识形态？是观念集合，是人们用来过滤实相，创造属于自己现实的滤镜。群氓意识形态会把文化分为高雅低俗、良好粗鄙、高级低级，主张把大众文化向高级良好的方向引导。文化民粹主义意识形态则抬高大众文化的价值，承认大众消费的主体创造性，反对精英对大众文化的霸权、收编。所谓霸权，就是掌权者通过重新解释历史弥合与被统治阶层之间分歧的能力。

意外之喜，是本书把最近 150 多年以来的欧洲哲学思潮做了个巡礼，可以从中看到马克思主义，精神分析，结构主义，女性主义，种族主义，后现代主义等等哲学思想在文化批判中的应用。

马克思主义则对工人阶级的文化非常认同，因而要么采取不评价只厘清的态度对待大众文化，要么对其大加赞扬；后马克思主义更是用霸权概念，将统治阶级修改文化表述、弥合阶级矛盾的做法加以批判。意识形态论，精神分析（语义网的勾连震颤），女性主义，种族主义等视角在马克思主义之后登上思想界舞台，纷纷重新分析、讲述、批判文化，进行反霸权的活动。后现代主义则解构这一切，把分析判断的主体归还个人。

拉康的精神分析是出书中捡到的宝，他对“实在界”，“想象界”，“象征界”的阐述精辟之极。但本书存在对拉康的严重误读。

所有的文化文本，包括本书、本评论，所有的文化实践，包括本书的书写出版发行，包括本评论的书写发布，都发生在拉康所说的“象征界”：没有任何真理在其中，且毫无意义。创造同时是破坏，理解同时是误读。

当作如是观。

### 02. [天津工人，1900—1949](https://neodb.social/book/4RFy8PAg8aYmvwxy1Gfqyx)

|  |  |  |
| --- | --- | --- |
| 2025-12-01 读过 | 历史 | 豆瓣评分 |

我给它【8 分】：八国联军 1900 年攻陷天津，对天津进行了两年的殖民统治，之后还是清朝统治。之后北洋政府从 1912 年到 1928 年，之后是南京十年，是国民政府时期。1937 年日本占领天津 8 年，到 1945 年日本投降。抗战胜利之后南京政府接管天津，但是国共内战到 1949 年。本书探讨的天津的工人状况，大的历史背景就是如此。但是作者行文之中，常常并未进行类似时期的划分。

作者 1979-1981 年在中国访谈搜集资料，1986 年成书发表于美国。2015 年翻译回中国。为天津城市历史作重要贡献。

天津九河下稍，交通便利，土地肥沃，商业发达。但在 1900 年左右资本，经济，行业都因租界割裂呈现破碎化。同时气候上，冬冻夏洪春风沙，秋天则可能闹蝗灾。底层人民生活困苦不堪。

书中详细考察了三种类型的工人形态：三条石的铸造业和机械制造业工人；运输搬运工人（脚行）；和大纱厂大工人和女工。

三条石的铸铁业和机器制造业起源于铁工匠为来往船只修复和制作生活铁器，后因仿制日本人的印刷机，织布机等机器起家。工人多是工头介绍来学徒，学满成匠人的方式，由外地招来。工人和厂主均来自农村，组织形式仍有很强的人情因素，工人吃住在厂，娱乐在附近的茶馆酒肆，挣钱回家，很少与本地人接触。类似当今的农民工。

运输业也即脚行的工人，有很强的地盘垄断，在地盘边界，常爆发暴力冲突。为了获得保护，工人常常必须加入帮派。脚行为青帮所垄断。书中对天津混混儿和后来的青帮做了些介绍。帮派势力强大到管理运输业的政府官员也是青帮的。脚行没有女工。

纱厂纺织业的厂规模，一般较大。招收女工，女工来源大部分是外地的。外地女工好管，便宜。一些技术活和体力活仍招男工。在大厂里的互助形式，男工流行拜把子结盟或入帮派，女性也拜干姐妹，或形成宗教小团体。男性很少结宗教团体的原因，大概是因为他们已经在社会上加入某种团体。

此书对工人们的娱乐文化生活也进行了详细总结。工人们多来自农村，各种节日习俗有很强的乡土情结。但在娱乐方式上，则往城市才有的方式靠近。

但书最终落脚关注点，仍是工人们的革命行动。日占时期，怠工，偷窃和破坏生产是工人对抗的主要形式。国民党时期，有工会，解决劳资纠纷。但其另一职责，就是防止共产党滲透。内战时期的许多罢工，则是有地下党发起的。但天津的工人关心自身的工作保障比关心政治更甚些。罢工的次数也比南方城市的少许多。

## 蘑菇

![](https://neodb.social/m/item/googlebooks/2025/09/04/3895ef17-2667-4ede-9c66-5e614e128178.jpg)

### 03. [Psilocybin Mushrooms in Their Natural Habitats](https://neodb.social/book/0uk66KdmPa5VSQ9KFt5DvW)

|  |  |  |
| --- | --- | --- |
| 2025-11-19 读过 | 真菌，蘑菇 | 豆瓣评分 |

我给它【9 分】：儿子很喜欢地雷，总在地上挖坑玩儿埋雷的游戏，于是就注意到了地上，会冒出蘑菇。于是又转而研究蘑菇，顺着他的兴趣，全家都开始读蘑菇书，看蘑菇视频，加入当地蘑菇协会，看蘑菇展，听蘑菇讲座，一同去采蘑菇，认蘑菇。在吃蘑菇上，全家态度是不一样的，儿子坚决不吃，其他人则享受蘑菇的美味。

致幻蘑菇，就在这样的学习中，作为知识进入视野。

本书是作者研究蘑菇有五十年的智慧结晶，前半部分有致幻蘑菇历史的梳理，又有种植方法、科学研究、应用前景、使用事项等的全面介绍；后半部分则是对致幻蘑菇属和一些含有致幻成分的其他属蘑菇的识别指南。识别指南非常详细，对每种蘑菇的外形特征，孢子特点，生活环境与分布、成分含量都做了介绍。可以说是致幻蘑菇一本通。

致幻蘑菇在世界上多数国家都是被列为毒品，不能流通交易。但在美国，写书传播使用种植的知识信息，则受宪法保护。美国的一些州已经允许商业种植、使用致幻蘑菇，并纳入严格的监管。有的国家公园会有禁止私人采摘致幻蘑菇的规定，并有执法者对违反规定的人罚款，成为类似查超速的政府“创收”。

政府的监管和限制，使得合法使用致幻蘑菇的花费非常贵。恐怕短时间内，许多人还会违法私自种植使用。

我自己对致幻蘑菇的看法：它呢，是自然大地安排给人的导师，教给人体认和经历人与他人、世界是合一的关系。它不是唯一能做到这一点的，但它直接，有效。唯一的缺点是，让有些人认为只能通过它才行。

后面的识别指南我没有读完。靠图片文字认识蘑菇，对我来说一是很难，二来也不算真正的认识。就像是，我不能说读过一个人的传记就认识他了。我期待能结识致幻蘑菇的熟人，然后被领着，去拜访它们，获得真正的认识。

### 04. [真菌改变人类文明史](https://neodb.social/book/1LjdlGyploLjpKBn6CwVbz)

|  |  |  |
| --- | --- | --- |
| 2025-11-30 读过 | 真菌，科普，蘑菇 | 豆瓣评分 |

我给它【8 分】：作者虽然是中国名，但讲蘑菇故事仍是西方视角。看了作者简介，果然是从西方受到的教育。

除了前言后记，本书共有五部分。

第一部分 介绍医疗或工业中用到的真菌应用，包括抗生素盘尼西林，预防器官移植排异反应的环孢霉素，服装生产中用到的纤维素酶以及用作杀虫剂的球孢白僵菌和绿色木霉。

第二部分介绍的则是降低各种农作物产量的致病真菌种群。几乎每种大规模农业，都受某种真菌侵害，玉米，大豆，小麦，稻米，咖啡，巧克力，红薯，无一放过。尤其是香蕉这种无性生殖品种，被真菌追杀到快要绝种。

第三部分则介绍致命或置换的蘑菇。致命蘑菇中最致命的 Death Cap。致幻蘑菇中的光盖伞，毒蝇伞和麦角菌，也有研究证明他们对某些心理疾病有治疗作用。

第四部分跟日常的吃有关。介绍了常见可食用的蘑菇，比如用作食物的各种超市里常见的洋菇，金针菇，平菇，杏鲍菇，香菇，木耳，竹荪，也有非常少见的松露和四大野生菌王（松茸，鸡油菌，牛肝菌，羊肚菌）；除此之外，有药用的冬虫夏草与灵芝；有日常做馒头蛋糕的酵母菌，各种酿酒，做奶酪，霉豆腐的菌种，都是丰富人们餐桌的。

第五部分介绍给我们赖以生存的环境和动物朋友们造成灭顶之灾的一些菌种。不过说起来，这些菌种的泛滥，也实在是跟人类在地球上乱窜，把一地之物种带到另一地，从而让致病菌种蔓延开来的缘故。

我们对真菌和蘑菇的所知，远还没有涵盖全部。期待有兴趣的人多多投入，研究真菌。

## 魔法世界

### 05. [哈利·波特与死亡圣器](https://neodb.social/book/3O8trrznirVQtORd17cPom)

|  |  |  |
| --- | --- | --- |
| 2025-11-07 读过 | 小说 | 豆瓣评分 9.1 |

我给它【9 分】：“阿瓦达索命咒” 对 “除你武器咒”，结束了伏地魔与哈利之间的命运纠结。

作为最强大的黑魔王，伏地魔的课题是征服死亡。他用了两个方法：1. 把自己的灵魂分裂，制造备份存在魂器，为的是一旦被杀死，可以通过魂器不断复活，而躲避死亡。2. 信奉魔法既强权的他，一直寻找最强不败的老魔杖，去打败威胁他存在的人。不会被打败，就不会死，也无需复活。如此的双保险最求，就是他的人生追求。其他人，不是帮手，就是绊脚石。哈利是最硬的那一块绊脚石。

魂器是逃避，圣器是对抗。二者皆未征服死亡。

邓布利多教给哈利，去直面死亡，去完完整整的经历。

伏地魔不知道，哈利承载了他的灵魂碎片，是他的魂器之一。他下手杀死哈利，也就击碎了自己的残魂。主动慨然赴死的哈利，反倒重获新生，成了死亡的征服者。

臣服与接纳，而非逃避会反抗，才是征服死亡的方法。

残魂破碎的伏地魔与重获新生的哈利，向对方发出魔咒。一个索命，一个夺刃。两个咒语都起了作用。作用在同一个人身上。正如最初的预言，只活一个。

哈利波特七部，出版时间 1997 到 2007，一共 11 年。书中，哈利从十一长到十七岁成年。若是十一岁的孩子追着哈利波特，一套书读完，正好二十一。虽然没能在 17 岁研究死亡的秘密，但 21 岁也不晚。

2025 年，距离哈利波特与死亡圣器的故事问世，过去 19 年了。应该是哈利的儿子阿不思 西弗内斯去霍格沃茨上学年头了。伤疤已经十九年没有疼过了，一切太平。

### 06. [神奇的魁地奇球](https://neodb.social/book/4DwNNTy5vIH6JO2lu0pNjx)

|  |  |  |
| --- | --- | --- |
| 2025-11-10 读过 | 哈利波特，小说，英国文学 | 豆瓣评分 7.7 |

我给它【7 分】：哈利波特系列衍生书。

哈利波特的故事，多发生在霍格沃茨魔法学校。魔法学校再神奇，主要目的也是上课。课业之余，学校生活中最值得期待的，大概就是魁地奇球比赛了。与赫敏学霸书虫不同，哈利并没有那么喜欢读书，遇到好老师学的就好点儿，遇到讨厌的老师学的就差。读书什么的，如果不是因为课业，不读都可以。读书那里比得上冒险和魁地奇。但关于魁地奇的书是例外，这大概是哈利唯一一本主动开读的书了。可见体育对于男孩子的吸引力。

魁地奇是一项风靡巫师界的团体对抗性运动。哈利一年级的时候就入队，成为找球手，后来又成长为队长。韦斯莱一家人，从乔治和弗雷德开始，几乎都进过哈利的队。他的队员们，最终也几乎都成了他最后对抗伏地魔之战的战友。哈利波特系列的每一部，都对魁地奇比赛的着墨甚多。

本书介绍了魁地奇运动的发展史，用球，规则，赛制，是怎样一步步的形成的。麻瓜们很容易读懂，因为读来像是足球，篮球，橄榄球的逸事史。
