## Zig 从 GitHub 迁移到 Codeberg

[![微软](https://icon.solidot.org/images/topics/topicMicrosoft.png?123)](/search?tid=150 "微软")

[Edwards](/~Edwards) (42866)发表于 2025年12月03日 22时50分 星期三 [新浪微博分享](//service.weibo.com/share/share.php?url=//www.solidot.org/story?sid=82959&appkey=1370085986&title=Zig%20%E4%BB%8E%20GitHub%20%E8%BF%81%E7%A7%BB%E5%88%B0%20Codeberg "新浪微博分享")
![](https://icon.solidot.org/images/a7c7.png)

**来自时间秘史**

Zig 软件基金会总裁兼首席开发者 Andrew Kelly 宣布项目将从 GitHub 迁移到 Codeberg，理由是微软对 AI 的迷恋正在毁掉 GitHub。他给出的一个例子是 GitHub Actions 的一个严重 bug 在 2022 年 2 月引入直到 2025 年 8 月才解决，该 bug 会导致死循环，导致进程一直运行其它任务无法执行。Andrew Kelly 认为 GitHub 不再致力于提供卓越的服务，用前 CEO 在离职前的“名言”——要么拥抱 AI 要么走——他们选择沉迷于 AI。Zig 不是唯一一个选择离开的项目，Dillo 浏览器项目也决定离开。与此同时，Codeberg 的支持会员数从 1 月的 600 人增加到了 1200 多人。
https://www.theregister.com/2025/12/02/zig\_quits\_github\_microsoft\_ai\_obsession/
