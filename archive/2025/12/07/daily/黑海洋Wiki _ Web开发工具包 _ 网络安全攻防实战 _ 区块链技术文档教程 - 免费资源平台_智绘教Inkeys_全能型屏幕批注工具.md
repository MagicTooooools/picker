## 智绘教Inkeys是什么？

智绘教Inkeys 是一款可以在屏幕任意角落自由书写与绘制的全能型屏幕批注工具，让创意直接落在屏幕之上。软件由 C++20 编写，专为 Windows 平台打造，前身为 Intelligent-Drawing-Teaching（简称 IDT），在保持轻量易用的同时提供强大的绘图与批注能力。

Inkeys 支持最低 Windows 7（RTM, sp0），兼容 32 位 / 64 位 / Arm64 等多种系统架构，为老旧设备和新款终端都预留了可靠的使用空间。

## 核心优势与功能亮点

### 开箱即用的简洁体验

下载解压即可使用，无需复杂安装和配置。
界面设计干净直观，新手用户也能快速上手，在课堂教学、演示汇报或日常讲解中迅速进入工作状态。

### 开源免费，更安心也更自由

项目在 GitHub 上开源，完全免费使用。
用户可以放心部署在个人电脑或办公环境中，开发者和高级用户也能基于开源代码进行学习与扩展。

### C++驱动的高性能绘图

Inkeys 注重性能优化，由 C++ 驱动，结合自研 UI 界面与绘图库，带来响应敏捷、书写顺滑的绘图体验。
无论是细致标注还是大幅面批注，都能保持流畅，减少卡顿与延迟感。

### 智能绘图与智能橡皮

软件提供多种智能绘图能力：

* 支持“抬笔拉直”和“停留拉直”功能，手绘直线更加自然
* 规划后续加入形状识别，让图形标注更省力

橡皮工具可以根据速度或压感动态调整粗细，更接近真实书写工具的反馈，擦除效果精细且高效。

### 多指触控与PPT联动

Inkeys 对多指操作提供全局支持，画笔、荧光笔、橡皮、形状等工具都可多指绘图，适配触控屏、手写屏、电纸板等多种输入设备。

在演示场景中，Inkeys 与 PowerPoint 和 WPS 有良好联动：

* 支持直接按键翻页，墨迹跟随页面切换
* 提供底栏翻页按钮，方便在讲课或演示时快速控制进度

### 超级置顶与广泛兼容

借助 UiAccess 技术，Inkeys 可以在多种系统界面上保持“超级置顶”状态，包括：

* 任务管理器（置顶模式）
* 任务栏和开始菜单
* 屏幕键盘
* 锁屏界面

在复杂系统环境下也能保证批注内容不被遮挡，提升教学和操作的连贯性。

### 兼容性与便携性

Inkeys 支持 Windows 7 RTM(sp0) 及以上版本系统，同时支持 x86 / x64 / Arm64（原生）架构。
单文件解压即可运行，适合放在 U 盘或网盘中随身携带，在教室、会议室、办公室都能快速启用。

## 版本说明：LTS与Insider通道

智绘教Inkeys 官方版本分为两类，满足不同用户需求：

* **正式版本 LTS**
  由开发者与公测用户共同验证，稳定性更高，适合日常教学、办公和正式场景长期使用。
* **预览版本 Insider**
  集成最新功能特性，由开发者 AlanCRL 直接推送，适合愿意体验新特性、并能接受少量不确定因素的用户。

程序内自动更新通道也分为“正式通道（LTS）”和“预览通道（Insider）”，用户可以依照使用场景自由选择。

第三方开发者提供的修改版和定制版不属于官方版本，但可以接入 Inkeys 的自动更新机制，并通过软件内修复功能恢复到官方版本。
