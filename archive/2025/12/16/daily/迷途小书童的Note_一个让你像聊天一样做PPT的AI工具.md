# [一个让你像聊天一样做PPT的AI工具](https://xugaoxiang.com/2025/12/16/banana-slides/)

你有没有遇到过这种情况：明天就要做汇报了，PPT 还是一片空白；脑子里想法一堆，却被排版、配色、找图这些琐事搞得心力交瘁？

别急，现在有一款叫 \*\*Banana Slides\*\* 的 AI 工具，能让你像聊天一样轻松做出又快又好看的PPT。它的口号是：\*\*“Vibe your PPT like vibing code”\*\*。

简单来说，Banana Slides 是一个\*\*原生 AI PPT 生成器\*\*，基于一个叫 \*\*nano banana pro\*\* 的强大 AI 模型。它不像市面上那些只能套模板、改改文字的“假AI”，而是真能理解你的想法，并按你的想法来生成、修改每一页PPT。

– \*\*一句话生成\*\*：你只要说“帮我做个关于预制菜生产线的 PPT”，它就能自动生成大纲、内容、配图，甚至排版都给你安排好。
– \*\*大纲模式\*\*：先写个提纲，AI 帮你扩充成完整的页面。
– \*\*页面描述\*\*：你可以直接告诉 AI 每一页要放什么内容、什么风格。

更厉害的是，\*\*你还能用自然语言修改\*\*！比如：“把第三页换成饼图”、“把标题颜色改成蓝色”、“加一个案例分析模块”……AI 会立刻响应，不用点来点去。

你可以上传 Word、PDF、Markdown、TXT 等格式的文档，AI 会自动：
– 提取文字重点
– 识别里面的图片或图表链接
– 甚至能参考你上传的设计图，模仿风格生成 PPT

这就避免了“图文割裂”“素材质量差”的老问题。

传统 PPT 工具得点菜单、拖图层、调参数……而 Banana Slides 让你\*\*用嘴说就行\*\*。不满意哪一块？框选它，然后说“换成柱状图”或者“文字放大一点”，AI 就会重新生成那一块，而且风格还跟整份 PPT 保持一致。

做完的 PPT 可以一键导出成 \*\*PPTX\*\* 或 \*\*PDF\*\* 格式，默认就是 16:9 的标准比例，排版完美，不用再手动调整，拿去开会、上课、汇报都行。

## 背后的技术

– \*\*前端\*\*：用 React + TypeScript 写成，界面流畅，支持拖拽、实时预览。
– \*\*后端\*\*：基于 Python Flask，用 SQLite 存数据，用 python-pptx 生成 PPT。
– \*\*AI 核心\*\*：调用 Google Gemini 或 OpenAI 的大模型（可通过 AIHubMix 平台接入），特别强调了对 \*\*nano banana pro 模型\*\*的优化支持 — 这个模型在图文生成上效果惊艳，能精准还原你描述的风格和内容。

## 部署步骤

推荐用 Docker 一键启动（适合大多数用户）

“`bash
git clone <https://github.com/Anionex/banana-slides>
cd banana-slides
cp .env.example .env
# 编辑 .env，填上你的 Gemini 或 OpenAI 的 API 密钥
docker compose up -d
“`

然后打开浏览器，访问 `http://localhost:3000` 就能用了！
