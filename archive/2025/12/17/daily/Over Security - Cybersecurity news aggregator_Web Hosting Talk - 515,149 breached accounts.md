# Web Hosting Talk Data Breach

## What Happened

In July 2016, [the Web Hosting Talk forum suffered a data breach that was subsequently listed for sale](https://web.archive.org/web/20160709135150/https%3A//www.csoonline.com/article/3093018/security/mac-forums-hot-scripts-and-web-hosting-talk-databases-for-sale.html). The breach of the vBulletin based forum exposed 515k user records including usernames, email addresses, IP addresses and salted MD5 password hashes.

Email addresses

IP addresses

Passwords

Usernames

## Recommended Actions

##### Change Your Password

If you haven’t already changed the password affected by this breach, do
so immediately on every account where it was used.

Wherever 2FA is supported, add an extra layer of security to your
