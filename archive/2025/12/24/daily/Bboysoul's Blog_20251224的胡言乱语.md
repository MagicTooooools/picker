1. ### Bichon

Bichon 是一款基于 Rust 构建的轻量级、高性能邮件归档工具,支持多账户 IMAP 同步、OAuth2 认证、全文检索、去重压缩存储,并提供现代化 WebUI 和完整 REST API,专为邮件归档和搜索而设计。

<https://github.com/rustmailer/bichon>

2. ### Building a High-End AI Desktop

从 7500 欧元购入二手 Grace-Hopper 服务器(2x H100 GPU + 1TB 内存),经历拆解、水冷改造、修复传感器故障等波折,最终打造出可在家运行 235B 参数模型的高性能 AI 桌面工作站的完整经历。

<https://dnhkng.github.io/posts/hopper/>

3. ### Traceroute Tool from Scratch in Go

从零开始用 Go 语言实现 traceroute 工具,详解 ICMP 协议、TTL 机制、网络层原理,并提供完整可运行代码,帮助理解数据包如何在互联网中跳转路由的全过程。

<https://kmcd.dev/posts/traceroute/>

4. ### Extracting JVM Data from Crash-Looping Java Containers

详细指南:如何从 Kubernetes 中崩溃循环的 Java 容器中提取 heap dump 和 JFR 数据,涵盖添加临时卷、修改 JVM 参数、使用调试容器等多种实战技巧,助力生产环境问题排查。

[https://medium.com/@zelldon91/getting-data-out-of-burning-java-containers-6e0c8bb53eec](https://medium.com/%40zelldon91/getting-data-out-of-burning-java-containers-6e0c8bb53eec)

5. ### Google De-Indexed My Entire Bear Blog

作者记录了在 Google Search Console 中误操作验证 RSS feed 后,整个 Bear 博客被 Google 完全取消索引的经历,尽管技术排查和 Bear 创始人支持均未找到原因,最终只能迁移到新域名。

<https://journal.james-zhan.com/google-de-indexed-my-entire-bear-blog-and-i-dont-know-why/>

6. ### My Favorite Self-Hosted Apps Launched in 2025

精选 2025 年发布的优秀自托管应用,包括 Docker 管理工具 Arcane、PDF 处理工具 BentoPDF、反向代理 Pangolin、笔记应用 NoteDiscovery、邮件归档工具等多个类别的实用项目。

<https://selfh.st/post/2025-favorite-new-apps/>

7. ### Podman Quadlet Volume Ownership

简明教程:在 Podman Quadlet 中使用 `:U` 卷挂载选项,让容器自动将源目录的所有权更改为容器内默认 UID/GID,解决 rootless 容器的权限问题。

<https://blog.khmersite.net/p/podman-quadlet-volume-ownership/>

8. ### 20 Years of Digital Life, Gone in an Instant

苹果资深开发者因使用礼品卡充值 iCloud 被永久封禁 Apple ID,损失数万美元设备和 20 年数字资产,客服拒绝提供原因和申诉渠道,展示了科技巨头对用户生杀大权的争议案例。

<https://hey.paris/posts/appleid/>

9. ### GraphQL: the enterprise honeymoon is over

基于多年企业级 Apollo 实践的反思:GraphQL 在已有 BFF 架构的企业环境中,其解决过度获取的核心优势被大幅削弱,而实现复杂度、可观测性、缓存脆弱性等代价却显著增加,多数场景下 REST 更实用。

<https://johnjames.blog/posts/graphql-the-enterprise-honeymoon-is-over>

10. ### Borg Web UI

Borg Web UI 是现代化的 Borg Backup 网页管理界面,提供零配置部署、实时备份进度监控、SSH 密钥管理、100+ 通知服务集成、前后备份脚本钩子等功能,支持多平台架构。

<https://github.com/karanhudia/borg-ui>
