# 智能可观测平台Dynatrace深化其AWS服务集成，新增多项AI与云原生监控功能

Dynatrace 已扩展其与亚马逊云服务（AWS）技术的集成，包括新增对 **Amazon Bedrock AgentCore**、AWS DevOps Agent 和 AWS Security Hub 的支持。

该公司还获得了新的 **AWS Agentic AI 专项认证**，彰显其在生产环境中观测和治理智能体 AI 系统的专业能力。这一进展使 Dynatrace 跻身首批获得此类认可的可观测性提供商之列，满足企业在部署高级 AI 工作负载时对更强监督和控制的需求。

### 奖项认可

Dynatrace 还被评为 **AWS 拉丁美洲（LATAM）地区公共部门技术合作伙伴年度奖**。该认可凸显了公司在支持 LATAM 地区政府、教育和非营利组织方面的工作，此前其在 EMEA 地区也获得过类似荣誉。

### 智能体 AI 聚焦

凭借 AWS Agentic AI 专项认证，Dynatrace 展示了在支持被称为“智能体系统”的复杂 AI 架构方面的技术能力和客户成功案例。这些系统通过协调多个自主智能体执行任务，需要增强的可见性、监控和治理才能实现大规模可靠运行。

“我们与 AWS 的合作持续推动创新，为客户带来切实成果，”Dynatrace 全球合作伙伴与联盟高级副总裁 Jay Snyder 表示，“获得 AWS Agentic AI 专项认证并被评为年度合作伙伴，突显了我们在 AI 时代赋能组织的共同承诺。Dynatrace 和 AWS 正携手帮助客户将云复杂性转化为战略优势，推动业务向前发展。”

### 扩展集成

Dynatrace 扩展的 AWS 集成涵盖多项新服务和现有服务：

1. 对 **Amazon Bedrock AgentCore** 的支持提供智能体系统的可观测性，允许客户在 AWS 环境中监控、调试和优化 AI 工作流。
2. 与 **AWS DevOps Agent** 的集成使用户能够加速根本原因分析，利用特定领域的 AWS 数据减少平均解决时间。Dynatrace 平台现在还能自动进行故障排除，从识别性能问题到提供修复说明。
3. 与 **AWS Security Hub** 的集成使 Dynatrace 用户能够加强其安全态势，获得实时可见性和 AI 驱动的洞察，以加速威胁检测并缩短响应时间。这旨在提高云环境中组织的弹性和合规性。

### Kiro 自主智能体

该公司还基于 Kiro 自主智能体推出了新功能，Kiro 利用 Dynatrace 的可观测性数据为开发人员提供协助。Kiro 帮助完成从错误分类到功能实现的各种任务，并设计为适合现有开发人员工作流。这种方法旨在通过让开发人员专注于更高层次的计划来支持更快的开发周期。

此外，Dynatrace 还推出了 Kiro power，这是一个为开发智能体提供 AI 驱动洞察的扩展。它在既定流程中为专业 AI 智能体提供性能分析，支持规范驱动的开发，无需上下文切换。

### 云运营改进

Dynatrace 的云运营解决方案现在包括对新 AWS 服务的自动发现，以及原生遥测和元数据摄入。由此产生的统一仪表板提供与性能和成本控制相关的 AI 驱动洞察，使团队能够实时了解复杂的云生态系统并支持现代化计划。

使用这些新集成和工具的客户可以管理云复杂性、提高运营效率并加强安全性和合规性。扩展产品的重点是让企业更容易采用和扩展 AI 驱动的工作负载，同时保持可靠性和监督。
