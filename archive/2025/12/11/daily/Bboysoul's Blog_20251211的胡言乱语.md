1. ### Requestly

Requestly 是一款本地优先的开源 API 客户端和 HTTP 拦截工具，结合了 Postman 和 Charles Proxy 的功能，支持 API 测试、模拟、环境管理和请求拦截修改，提供浏览器扩展和桌面应用，并支持从 Postman、Insomnia 等工具一键导入配置。

<https://github.com/requestly/requestly>

2. ### Modernizing Reddit’s Comment Backend Infrastructure

Reddit 工程团队详细分享了如何将评论系统从 Python 单体架构迁移到 Go 微服务的完整过程，包括通过"双写+镜像数据库"实现安全的 tap compare 测试、解决跨语言序列化和数据库竞态问题，最终在零中断的情况下完成迁移并将 p99 延迟降低了一半。

<https://www.reddit.com/r/RedditEng/comments/1mbqto6/modernizing_reddits_comment_backend_infrastructure/>

3. ### Hidden Microphone on NanoKVM

安全研究员在中国智能 KVM 设备 NanoKVM 中发现了一个未在文档中明确说明的内置麦克风，配合设备内置的录音工具、默认密码、与中国服务器通信以及预装的黑客工具等安全漏洞，引发了对硬件隐私和供应链安全的严重担忧。

<https://telefoncek.si/2025/02/2025-02-10-hidden-microphone-on-nanokvm/>

4. ### Screenshots from developers: 2002 vs. 2015

作者在 2002 年收集了一批 Unix 开发者的桌面截图，13 年后重新联系这些传奇程序员（包括 Brian Kernighan、Richard Stallman、Bram Moolenaar 等），对比他们的工作环境变化，展现了从简约终端到现代 GUI 的演变，以及开发者们对命令行工具的持久热爱。

[https://anders.unix.se/2015/12/10/screenshots-from-developers–2002-vs.-2015/](https://anders.unix.se/2015/12/10/screenshots-from-developers--2002-vs.-2015/)

5. ### SourceGit

SourceGit 是一款跨平台（Windows/macOS/Linux）的开源 Git GUI 客户端，支持可视化提交图、SSH 访问、GitFlow、交互式变基、子模块、工作树等功能，并内置 AI 生成提交信息、明暗主题切换和多语言支持，提供简洁高效的 Git 操作体验。

<https://github.com/sourcegit-scm/sourcegit>

6. ### 腾讯 UGit

腾讯自研的 Git 客户端 UGit，专为大型项目和游戏开发优化，提供类 SVN 的集中式体验、便捷的 LFS 大文件管理、快速提交（无需频繁更新）、工蜂锁协作、子目录检出、仓库同步、多仓库管理等特色功能，支持 Windows 和 macOS。

<https://ugit.qq.com/zh/>

7. ### Hunting for North Korean Fiber Optic Cables

通过分析公开资料、卫星图像、铁路沿线照片和网络追踪数据，研究者推测朝鲜光纤网络可能沿东海岸铁路线从俄罗斯边境经清津、咸兴到平壤，文章详细探讨了朝鲜网络基础设施布局、与俄罗斯和中国的连接方式，以及通过延迟测试推断服务器位置的方法。

<https://nkinternet.com/2025/12/08/hunting-for-north-korean-fiber-optic-cables/>

8. ### Size of Life

Neal Agarwal 制作的交互式可视化网站，通过精美插图和动画展示从微观病毒、细菌到宏观动植物的相对大小关系，让用户以直观方式理解生命体的尺度差异，提供沉浸式的科学教育体验。

<https://neal.fun/size-of-life/>

9. ### Making macOS Bearable

作者详细分享了如何将 macOS 从默认的"视觉搜索"工作流改造为高效的"肌肉记忆"系统，通过 Aerospace 平铺窗口管理器、Tmux root 绑定、模态编辑器 Helix、Vimium 浏览器插件和 Homerow 系统级快捷键，实现了类似 Arch Linux 的无缝工作体验。

<https://seg6.space/posts/making-macos-bearable/>

10. ### Zerobyte

Zerobyte 是基于 Restic 的开源备份自动化工具，提供现代化 Web 界面管理加密备份，支持 NFS、SMB、WebDAV、本地目录和 40+ 云存储（通过 rclone），具备灵活的调度策略、端到端加密、保留策略和 Docker volume 插件功能。

<https://github.com/nicotsx/zerobyte>
