### **EdgeжөҸи§ҲеҷЁзҡ„ж— еЈ°еҸҳйқ©пјҡеҫ®иҪҜеҰӮдҪ•е°Ҷе…¶йҮҚеЎ‘дёәдәәе·ҘжҷәиғҪжҢҮжҢҘдёӯеҝғ**

й•ҝжңҹд»ҘжқҘпјҢеҫ®иҪҜж——дёӢзҡ„ Edge жөҸи§ҲеҷЁе§Ӣз»ҲдҪҚеұ…и°·жӯҢ Chrome зӯүиЎҢдёҡе·ЁеӨҙд№ӢеҗҺпјҢдҪҶиҝ‘жңҹзҡ„дёҖзі»еҲ—еҚҮзә§дёҫжҺӘйў„зӨәзқҖдёҖеңәеӨ§иғҶиҪ¬еһӢ вҖ”вҖ” е®ғжңүжңӣйҮҚж–°е®ҡд№үиҮӘиә«еңЁж•°еӯ—з”ҹжҖҒзі»з»ҹдёӯзҡ„и§’иүІгҖӮеңЁиҝҮеҺ»ж•°жңҲй—ҙпјҢеҫ®иҪҜжӮ„з„¶е°Ҷ Copilot дәәе·ҘжҷәиғҪеҠ©жүӢзҡ„ж ёеҝғеҠҹиғҪж•ҙеҗҲиҮі Edge жөҸи§ҲеҷЁпјҢдҪҝе…¶д»ҺдёҖж¬ҫеҚ•зәҜзҡ„зҪ‘йЎөжөҸи§Ҳе·Ҙе…·пјҢиң•еҸҳдёәе…·еӨҮжҷәиғҪдәӨдә’иғҪеҠӣгҖҒеҸҜдё»еҠЁжҸҗдҫӣжңҚеҠЎзҡ„дјҷдјҙгҖӮиҝҷдёҖиҪ¬еҸҳз»қйқһжөҒдәҺиЎЁйқўзҡ„и§Ҷи§үжӣҙж–°пјҢиҖҢжҳҜеҜ№з”ЁжҲ·зҪ‘йЎөжөҸи§Ҳж–№ејҸзҡ„ж №жң¬жҖ§йҮҚжһ„пјҡе®ғе°Ҷдәәе·ҘжҷәиғҪжҠҖжңҜдёҺж—ҘеёёзҪ‘йЎөеҜјиҲӘж·ұеәҰиһҚеҗҲпјҢе…¶иғҢеҗҺжҠҳе°„еҮәеҫ®иҪҜжҺЁеҠЁдәәе·ҘжҷәиғҪй©ұеҠЁеһӢи®Ўз®—зҡ„е®ҸеӨ§ж„ҝжҷҜгҖӮ

иҝҷеңәеҸҳйқ©зҡ„ж ёеҝғй©ұеҠЁеҠӣпјҢжәҗдәҺеҫ®иҪҜе°Ҷ Copilot е…ЁйқўеөҢе…Ҙж——дёӢе…Ёзі»дә§е“Ғзҡ„йЎ¶еұӮжҲҳз•ҘгҖӮжҚ®иЎҢдёҡи§ӮеҜҹдәәеЈ«йҖҸйңІпјҢEdge жөҸи§ҲеҷЁзҡ„жңҖж–°йў„и§ҲзүҲжң¬е·ІйҮҮз”ЁдёҺ Copilot еҗҢжәҗзҡ„и®ҫи®ЎиҜӯиЁҖпјҢеңЁиүІеҪ©жҗӯй…ҚгҖҒеӯ—дҪ“йҖүжӢ©е’Ңз•ҢйқўеёғеұҖдёҠдҝқжҢҒй«ҳеәҰдёҖиҮҙгҖӮиҝҷе№¶йқһдёҖи№ҙиҖҢе°ұзҡ„еҪ»еә•йқ©ж–°пјҢиҖҢжҳҜеҫӘеәҸжёҗиҝӣзҡ„иҝӯд»ЈеҚҮзә§ вҖ”вҖ” д»ҺйҮ‘дёқйӣҖпјҲCanaryпјүе’ҢејҖеҸ‘иҖ…пјҲDevпјүжөӢиҜ•йў‘йҒ“зҡ„жӣҙж–°еҚіеҸҜзӘҘи§Ғз«ҜеҖӘпјҡжөӢиҜ•з”ЁжҲ·еҸ‘зҺ°пјҢжөҸи§ҲеҷЁзҡ„и®ҫзҪ®йЎөйқўдёҺеҸій”®иҸңеҚ•еқҮз»ҸиҝҮйҮҚж–°и®ҫи®ЎпјҢи§Ҷи§үйЈҺж јдёҺ Copilot еә”з”ЁеҰӮеҮәдёҖиҫҷгҖӮ

иҝҷдәӣеҸҳйқ©зҡ„ж ёеҝғпјҢжҳҜеҫ®иҪҜеҠӣжҺЁзҡ„ \*\*вҖңжҷәиғҪд»ЈзҗҶејҸи®Ўз®—вҖқ\*\* зҗҶеҝөгҖӮеңЁиҝҷдёҖжЁЎејҸдёӢпјҢдәәе·ҘжҷәиғҪдёҚеҶҚиў«еҠЁе“Қеә”з”ЁжҲ·жҢҮд»ӨпјҢиҖҢжҳҜиғҪеӨҹдё»еҠЁйў„еҲӨйңҖжұӮе№¶иҮӘеҠЁе®ҢжҲҗд»»еҠЎгҖӮEdge жөҸи§ҲеҷЁзҡ„ Copilot жЁЎејҸжӯЈжҳҜиҝҷдёҖзҗҶеҝөзҡ„е…ёиҢғпјҡе®ғж”ҜжҢҒжөҸи§ҲеҷЁеӨ„зҗҶеӨҡжӯҘйӘӨж“ҚдҪңгҖҒз®ЎзҗҶж ҮзӯҫйЎөпјҢз”ҡиҮіиғҪи·ЁеӨҡдёӘе·Іжү“ејҖйЎөйқўиҝӣиЎҢйҖ»иҫ‘еҲҶжһҗдёҺжҺЁзҗҶгҖӮдҫӢеҰӮпјҢз”ЁжҲ·еҸҜжҢҮд»Ө Copilot жұҮжҖ»еӨҡиҫҫ 30 дёӘж ҮзӯҫйЎөзҡ„еҶ…е®№пјҢжҲ–и®©жөҸи§ҲеҷЁиҮӘдё»и®ҝй—®еҗ„зұ»зҪ‘з«ҷпјҢиҮӘеҠЁе®ҢжҲҗиЎҢзЁӢйў„и®ўзӯүе·ҘдҪңжөҒгҖӮ

### **EdgeжөҸи§ҲеҷЁдәәе·ҘжҷәиғҪеҠҹиғҪж•ҙеҗҲзҡ„зјҳиө·**

иҝҷеңәиң•еҸҳе§ӢдәҺ 2025 е№ҙжң«пјҢеҪјж—¶еҫ®иҪҜеҸ‘еёғзҡ„е…¬е‘Ҡе°ұе·ІйҖҸйңІеҮәејәеҢ–жөҸи§ҲеҷЁдәәе·ҘжҷәиғҪиғҪеҠӣзҡ„дҝЎеҸ·гҖӮ2025 е№ҙ 10 жңҲпјҢеҫ®иҪҜжӯЈејҸжҺЁеҮә Edge жөҸи§ҲеҷЁ Copilot жЁЎејҸпјҢе°Ҷе…¶е®ҡдҪҚдёәдёҖж¬ҫ \*\*вҖңдәәе·ҘжҷәиғҪжөҸи§ҲеҷЁвҖқ\*\*пјҢе®һзҺ°дәҶд»Һиў«еҠЁе·Ҙе…·еҲ°дё»еҠЁеҚҸдҪңдјҷдјҙзҡ„и§’иүІиҪ¬еҸҳгҖӮжҚ®еҫ®иҪҜ Edge е®ҳж–№еҚҡе®ўзҡ„ж–Үз« иҜҰиҝ°пјҢиҜҘеҠҹиғҪж”ҜжҢҒиҜӯйҹіеҜјиҲӘгҖҒиҮӘеҠЁеҢ–ж“ҚдҪңе’ҢеңәжҷҜеҢ–жөҸи§Ҳи·Ҝеҫ„и§„еҲ’пјҢд»Һж №жң¬дёҠж”№еҸҳдәҶз”ЁжҲ·дёҺеңЁзәҝеҶ…е®№зҡ„дәӨдә’ж–№ејҸгҖӮ

дёҡеҶ…дәәеЈ«жҢҮеҮәпјҢжӯӨж¬ЎеҚҮзә§жҳҜеңЁж—©жңҹеҠҹиғҪж•ҙеҗҲзҡ„еҹәзЎҖдёҠиҝӯд»ЈиҖҢжқҘ вҖ”вҖ” ж—©еңЁ 2023 е№ҙпјҢеҫ®иҪҜе°ұе·Ідёә Edge жөҸи§ҲеҷЁдҫ§иҫ№ж Ҹж·»еҠ дәҶ Microsoft 365 Copilot зҡ„и®ҝй—®е…ҘеҸЈгҖӮиҖҢжӯӨж¬Ўз”ЁжҲ·з•Ңйқўзҡ„е…ЁйқўйҮҚеЎ‘пјҢе Әз§°ж Үеҝ—жҖ§зҡ„иҪ¬жҠҳзӮ№гҖӮжҚ®з§‘жҠҖеӘ’дҪ“*Windows Central*жҠҘйҒ“пјҢеңЁ Edge йў„и§ҲзүҲжң¬дёӯпјҢжөҸи§ҲеҷЁз•ҢйқўеӨ§йҮҸеҖҹйүҙ Copilot зҡ„и®ҫи®Ўе…ғзҙ пјҢеҢ…жӢ¬е…Ёж–°ж”№зүҲзҡ„ж–°ж ҮзӯҫйЎөпјҢд»ҘеҸҠз»ҸиҝҮдјҳеҢ–зҡ„иҸңеҚ•еёғеұҖ вҖ”вҖ” дёҺдј з»ҹжөҸи§ҲжҺ§еҲ¶еҠҹиғҪзӣёжҜ”пјҢ**дәәе·ҘжҷәиғҪй©ұеҠЁзҡ„жҙһеҜҹеҠҹиғҪиў«зҪ®дәҺдјҳе…Ҳең°дҪҚ**гҖӮ

з„¶иҖҢпјҢжү№иҜ„иҖ…иҙЁз–‘жӯӨдёҫз©¶з«ҹжҳҜеҲӣж–°зӘҒз ҙиҝҳжҳҜиҝҮеәҰи¶Ҡз•ҢгҖӮйғЁеҲҶз”ЁжҲ·еҜ№ж— жі•е…ій—ӯиҝҷдәӣеҠҹиғҪеҸҳжӣҙзҡ„и®ҫи®ЎиЎЁзӨәдёҚж»Ў вҖ”вҖ” жңүз”ЁжҲ·еҸҚйҰҲпјҢеҚідҪҝе…ій—ӯ Copilot жЁЎејҸпјҢж–°зүҲз”ЁжҲ·з•Ңйқўд»Қдјҡдҝқз•ҷгҖӮиҝҷдёҖй—®йўҳеңЁзӨҫдәӨе№іеҸ° X дёҠеј•еҸ‘зғӯи®®пјҡз§‘жҠҖзҲұеҘҪиҖ…д»¬дёҖж–№йқўжӢ…еҝ§иҪҜд»¶дјҡж„ҲеҸ‘иҮғиӮҝгҖҒз”ЁжҲ·дјҡиў«ејәеҲ¶дҪҝз”Ёдәәе·ҘжҷәиғҪеҠҹиғҪпјҢеҸҰдёҖж–№йқўд№ҹи®ӨеҸҜе…¶еңЁжҸҗеҚҮе·ҘдҪңж•ҲзҺҮж–№йқўзҡ„жҪңеҠӣгҖӮ

### **еӨҮеҸ—е®Ўи§Ҷзҡ„з”ЁжҲ·дҪ“йӘҢ**

ж·ұе…ҘеҲҶжһҗиҝҷдәӣеҸҳйқ©еҜ№з”ЁжҲ·дҪ“йӘҢзҡ„еҪұе“ҚпјҡEdge жөҸи§ҲеҷЁжӯӨж¬ЎеҖҹйүҙ Copilot зҡ„з•ҢйқўйҮҚеЎ‘пјҢж—ЁеңЁи®©жөҸи§ҲиҝҮзЁӢжӣҙзӣҙи§ӮгҖҒжӣҙй«ҳж•ҲгҖӮй’ҲеҜ№е•ҶеҠЎз”ЁжҲ·пјҢжҷәиғҪд»ЈзҗҶжЁЎејҸеҸҜиҮӘеҠЁе®ҢжҲҗж•°жҚ®жұҮжҖ»зӯүйҮҚеӨҚжҖ§д»»еҠЎпјҢиғҪжҳҫи‘—иҠӮзңҒиҒҢеңәе·ҘдҪңж—¶й—ҙгҖӮеҫ®иҪҜе®ҳж–№еӯҰд№ е№іеҸ° Microsoft Learn зҡ„ж–ҮжЎЈиҜҰз»ҶеҲ—еҮәдәҶе•Ҷз”ЁзҺҜеўғдёӢзҡ„й…ҚзҪ®ж–№жЎҲпјҢзқҖйҮҚејәи°ғдәҶиҝҷдәӣдәәе·ҘжҷәиғҪеўһејәеҠҹиғҪеңЁ**е®үе…ЁжҖ§дёҺеҗҲи§„жҖ§**ж–№йқўзҡ„и®ҫи®ЎгҖӮ

дҪҶз”ЁжҲ·еҸҚйҰҲе№¶йқһдёҖзүҮеҸ«еҘҪгҖӮ2025 е№ҙ 11 жңҲпјҢ*Windows Central*еҸ‘иЎЁж–Үз« пјҢеҜ№ Edge дҫ§иҫ№ж ҸеҠҹиғҪеҸҜиғҪиў«еүҠејұзҡ„зҺ°зҠ¶иЎЁзӨәжғӢжғңпјҢе№¶и®ӨдёәиҝҷжҳҜ Copilot еҠҹиғҪжҢҒз»ӯжү©еј зҡ„еҗҺжһңгҖӮиҝҷдёҖзҺ°иұЎжҠҳе°„еҮәдёҖдёӘжҷ®йҒҚзҹӣзӣҫпјҡдәәе·ҘжҷәиғҪж•ҙеҗҲиҷҪиғҪеёҰжқҘдҫҝеҲ©пјҢеҚҙд№ҹеҸҜиғҪз–ҸиҝңйӮЈдәӣеҒҸеҘҪиҪ»йҮҸеҢ–жөҸи§ҲеҷЁгҖҒдёҚж„ҝиў«дәәе·ҘжҷәиғҪеҠҹиғҪйў‘з№Ғе№Іжү°зҡ„з”ЁжҲ·гҖӮ

д»ҺжҠҖжңҜеұӮйқўжқҘзңӢпјҢжӯӨж¬ЎеҚҮзә§дҫқжүҳдәҺ GPT-5 зӯүе…ҲиҝӣжЁЎеһӢзҡ„иғҪеҠӣжҸҗеҚҮ вҖ”вҖ” иҝҷдёҖзӮ№еңЁ 2025 е№ҙ 11 жңҲеҫ®иҪҜ Copilot е®ҳж–№еҚҡе®ўеҸ‘еёғзҡ„ Copilot Studio жӣҙж–°еҶ…е®№дёӯе·ІжңүдҪ“зҺ°гҖӮжҚ® Windows йў„и§ҲдҪ“йӘҢи®ЎеҲ’пјҲWindows Insiderпјүзҡ„жңҖж–°еҠЁжҖҒпјҢеҠҹиғҪж•ҙеҗҲеҗҺпјҢEdge жөҸи§ҲеҷЁеҸҜе®һзҺ°жӣҙеӨҚжқӮзҡ„д»»еҠЎпјҢдҫӢеҰӮеңЁ Windows зі»з»ҹдёӯзӣҙжҺҘзј–иҫ‘ж–Үжң¬пјҢжҲ–д»Һд»»еҠЎж ҸдёҖй”®еҲҶдә«еә”з”ЁзӘ—еҸЈгҖӮ

### **еҜ№еҫ®иҪҜз”ҹжҖҒзі»з»ҹзҡ„жҲҳз•Ҙж„Ҹд№ү**

еҫ®иҪҜеҜ№ Edge жөҸи§ҲеҷЁзҡ„ж”№йҖ пјҢжҳҜе…¶дәәе·ҘжҷәиғҪжҠҖжңҜе…Ёйқўжё—йҖҸе…Ёзі»дә§е“Ғзҡ„зј©еҪұгҖӮжӯӨеүҚпјҢеҫ®иҪҜе°Ҷ Office еҠһе…¬еҘ—д»¶жӣҙеҗҚдёә Microsoft 365 CopilotпјҢе°Ҫз®ЎиҝҷдёҖдёҫжҺӘеӣ йҡҗз§Ғй—®йўҳйҒӯеҲ°з”ЁжҲ·еҸҚеҜ№пјҲз§‘жҠҖеӘ’дҪ“*WebProNews*жӣҫеҜ№жӯӨиҝӣиЎҢжҠҘйҒ“пјүпјҢдҪҶд№ҹеҮёжҳҫдәҶеҫ®иҪҜе°Ҷдәәе·ҘжҷәиғҪдҪңдёә**ж ёеҝғе·®ејӮеҢ–з«һдәүеҠӣ**зҡ„еҶіеҝғгҖӮеңЁ Edge жөҸи§ҲеҷЁдёҠпјҢиҝҷдёҖжҲҳз•ҘдҪ“зҺ°дёәпјҡжөҸи§ҲеҷЁдёҚеҶҚеұҖйҷҗдәҺзҪ‘йЎөжөҸи§ҲеҠҹиғҪпјҢиҖҢжҳҜе…·еӨҮдәҶйў„еҲӨйңҖжұӮгҖҒеҲҶжһҗдҝЎжҒҜгҖҒжү§иЎҢж“ҚдҪңзҡ„иғҪеҠӣ вҖ”вҖ” иҝҷжҲ–е°ҶдҪҝе…¶жҲҗдёәжүҝиҪҪеҫ®иҪҜдәәе·ҘжҷәиғҪйӣ„еҝғзҡ„ж ёеҝғжһўзәҪгҖӮ

еёӮеңәз«һдәүеҺӢеҠӣеҗҢж ·жҳҜйҮҚиҰҒжҺЁжүӢгҖӮйқўеҜ№и°·жӯҢдёә Chrome жөҸи§ҲеҷЁж•ҙеҗҲ Gemini дәәе·ҘжҷәиғҪзҡ„з«һдәүжҖҒеҠҝпјҢеҫ®иҪҜе·ІдёҚе®№жңүеӨұгҖӮEdge зҡ„ Copilot жЁЎејҸеҮӯеҖҹеӨҡж ҮзӯҫйЎөжҷәиғҪеҲҶжһҗе’ҢиҮӘеҠЁеҢ–е·ҘдҪңжөҒдёӨеӨ§зү№иүІпјҢиөӢдәҲдәҶиҮӘиә«зӢ¬зү№зҡ„з«һдәүдјҳеҠҝ вҖ”вҖ” жҒ°еҰӮе…¶еҗҚдёӯзҡ„ вҖңEdgeвҖқпјҲжңү вҖңдјҳеҠҝвҖқ д№Ӣж„ҸпјүпјҢе®ғи®©жөҸи§ҲеҷЁдёҚеҶҚжҳҜдёҖдёӘйқҷжҖҒзҡ„зҪ‘йЎөе…ҘеҸЈпјҢиҖҢжҳҜиғҪзІҫеҮҶжҚ•жҚүз”ЁжҲ·ж„Ҹеӣҫзҡ„еҠҹиғҪе»¶дјёгҖӮ

д»Һдә§е“ҒејҖеҸ‘и§’еәҰзңӢпјҢиҝҷдәӣеҠҹиғҪжӯЈд»Ҙиҝӯд»Јж–№ејҸйҖҗжӯҘжҺЁйҖҒгҖӮз§‘жҠҖеӘ’дҪ“*The Verge*зҡ„жҠҘйҒ“иҒҡз„ҰдәҺжӯӨж¬ЎеӨ§и§„жЁЎз”ЁжҲ·з•ҢйқўйҮҚжһ„пјҢжҢҮеҮә Edge жөҸи§ҲеҷЁжӯЈеҖҹйүҙ Copilot зҡ„и§Ҷи§үйЈҺж јпјҢж—ЁеңЁжү“йҖ еҫ®иҪҜе…Ёзі»еҲ—еә”з”Ёзҡ„з»ҹдёҖдҪ“йӘҢгҖӮиҝҷдёҖдёҫжҺӘжҲ–е°Ҷдёә \*\*вҖңжҷәиғҪд»ЈзҗҶејҸ Windows зі»з»ҹвҖқ\*\* й“әе№ійҒ“и·Ҝ вҖ”вҖ” жңӘжқҘпјҢдәәе·ҘжҷәиғҪд»ЈзҗҶе°ҶиғҪж— зјқеӨ„зҗҶеҗ„зұ»еӨҚжқӮд»»еҠЎгҖӮ
