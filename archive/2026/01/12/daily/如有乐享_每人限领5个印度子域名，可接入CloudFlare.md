## 使用限制

每个账号可注册5个子域名

## 注册账号

* 点击“开始使用”或“注册”。
* 请输入您的姓名、电子邮件地址和安全密码。
* 完成安检（旋转闸门）。
* 点击“创建账户”。
* 您将看到验证页面。请查看您的电子邮件，您会收到一个6位数的验证码。
* 请输入验证码以验证您的账户。

**目前 GitHub 已暂停接受新用户注册。**要使用 GitHub 登录，您必须**先创建一个邮箱/密码账户**（使用与您的 GitHub 账户相同的邮箱地址）。

注册完成后，您可以使用该邮箱地址登录并关联您的 GitHub 账户。

![](https://51.ruyo.net/wp-content/themes/CorePress-Pro/static/img/loading/doublering.svg)

## 简要操作

1，账号注册成功后，可输入要注册的域名前缀

2，如图提示绿色说明该子域名可以注册

![](https://51.ruyo.net/wp-content/themes/CorePress-Pro/static/img/loading/doublering.svg)

3，注册成功后，可前往CloudFlare添加该域名。按步骤提示修改该子域名的NS信息即可！

![](https://51.ruyo.net/wp-content/themes/CorePress-Pro/static/img/loading/doublering.svg)

4，修改NS后稍等一会就生效了！

![](https://51.ruyo.net/wp-content/themes/CorePress-Pro/static/img/loading/doublering.svg)

## 关于续期

子域名到期前60天可点击【续订】按钮免费续订一年！不会收取任何费用！

免费域名千万不要用于线上或者重要服务。随时可能翻车！
