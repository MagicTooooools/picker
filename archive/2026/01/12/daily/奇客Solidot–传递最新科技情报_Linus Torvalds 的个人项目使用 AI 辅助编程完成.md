**来自穿越时空的少女**

Linus Torvalds 在 GitHub 上公开了名为 AudioNoise 的随机数字音效个人项目，他在 README 文件中透露使用了 Google 的 AI 驱动 IDE 工具 Antigravity 进行了辅助编程——或者叫 Vibe Coding。Antigravity 是 Windsurf 的分支，而 Windsurf 则是微软 VS Code 的分支。
