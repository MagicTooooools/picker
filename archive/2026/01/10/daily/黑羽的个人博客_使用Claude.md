Claude е®үиЈ…е®ҳж–№[ж–ҮжЎЈ](https://code.claude.com/docs/zh-CN/overview#native-install-recommended)е·Із»ҸеҫҲиҜҰз»ҶдәҶпјҢиҝҳжңүдёӯж–ҮгҖӮ
дёҚиҝҮеӨ§йҷҶз”ЁжҲ·жҳҜз”ЁдёҚдәҶе®ғд»¬зҡ„жңҚеҠЎзҡ„пјҢеҸҜд»ҘдҪҝз”Ёд»ЈзҗҶз«ҷпјҢдҫӢеҰӮ [Openroute](https://openrouter.ai/)жҲ–иҖ…е…¶д»–гҖӮ
Openroute зҡ„[ж–ҮжЎЈ](https://openrouter.ai/docs/guides/guides/claude-code-integration)йҮҢжңүиҜҙжҳҺпјҢеҸҜд»Ҙж·»еҠ дёӢйқўзҡ„зҺҜеўғеҸҳйҮҸжқҘдҪҝз”Ёд»–д»¬зҡ„д»ЈзҗҶгҖӮ
```shell
# Set these in your shell (e.g., ~/.bashrc, ~/.zshrc)
export ANTHROPIC\_BASE\_URL="https://openrouter.ai/api"
export ANTHROPIC\_AUTH\_TOKEN="$OPENROUTER\_API\_KEY"
export ANTHROPIC\_API\_KEY=""
```
дёҚиҝҮиҝҳжјҸжҺүдёҖдёӘпјҢз”ұдәҺ Claude еҗҜз”ЁеҲқе§ӢеҢ–зҡ„ж—¶еҖҷиҰҒиҝһжҺҘе®ғд»¬зҡ„жңҚеҠЎеҷЁпјҢжҲ‘д»¬еңЁиҝҷдёҖжӯҘе°ұдјҡиў«жӢ’з»қпјҢж— жі•иҝӣиЎҢдёӢеҺ»пјҢжүҖд»ҘйңҖиҰҒеңЁз”ЁжҲ·зӣ®еҪ•ж·»еҠ  `.claude.json` ж–Үд»¶пјҢеҰӮжһңе·Із»ҸжңүдәҶе°ұзӣҙжҺҘдҝ®ж”№пјҢж·»еҠ еҰӮдёӢеҸӮж•°пјҡ
```json
{
"hasCompletedOnboarding": true
}
```
ж„ҸжҖқжҳҜе‘ҠиҜүClaudeпјҢжҲ‘д»¬е·Із»Ҹе®ҢжҲҗдәҶеҲқе§ӢеҢ–пјҢз»§з»ӯеҗҺйқўзҡ„гҖӮ
Openroute зҡ„жЁЎеһӢд№ҹжҳҜдёҚдҫҝе®ңпјҢжүҖд»ҘжҲ‘з”ЁдәҶеӯ—иҠӮзҒ«еұұзҡ„иҜ•иҜ•пјҢйҰ–жңҲ9еқ—й’ұгҖӮдёҚиҝҮзӣ®еүҚж„ҹи§үиҙЁйҮҸдёҚеӨӘиЎҢгҖӮ
> дёӢйқўзҡ„йЎ№зӣ®е°ұжҳҜз”Ёе®ғдҝ®ж”№зҡ„пјҢжҲ‘ж•ҙдәҶеҚҠеӨ©пјҢиҝһдёӘ Dockerfile йғҪиғҪжҗһй”ҷгҖӮ
![image.png](https://thetbw-hk.cos.thetbw.xyz/blog/image\_1768045385494.png)
Claude дјҡеё®дҪ е®ҢжҲҗеҫҲеӨҡж“ҚдҪңпјҢеҢ…жӢ¬жү§иЎҢжһ„е»әе‘Ҫд»ӨпјҢдҪҶжҳҜдёҮдёҖ AI зҠҜз—…дәҶпјҢеҸҜиғҪжҠҠдҪ еҪ“еүҚз”өи„‘зҺҜеўғжҗһзҡ„дёҖеӣўзіҹпјҢжүҖжңүиҝҳжңүдёӘж–№жі•жҳҜеңЁ Docker дёӯиҝҗиЎҢпјҢжҢӮиҪҪеҪ“еүҚйЎ№зӣ®еҲ° Docker дёӯгҖӮ
Github дёӯзңӢеҲ°дёҖдёӘйЎ№зӣ® [claude-docker](https://github.com/VishalJ99/claude-docker)пјҢдёҚиҝҮиҝҷдёӘеҸӘж”ҜжҢҒ MacпјҢиҝҳйў„иЈ…дәҶдёҖдәӣ MCPпјҢжҲ‘ж„ҹи§үдёҚйңҖиҰҒпјҢе°ұеҲ дәҶпјҢиҝҳеўһеҠ дәҶ Linux ж”ҜжҢҒпјҢ
йЎ№зӣ®ең°еқҖ[еңЁиҝҷ](https://github.com/thetbw/claude-docker)гҖӮ
дҪҝз”Ёж–№жі•еҰӮдёӢ
```shell
# 1. е…ӢйҡҶйЎ№зӣ®
git clone https://github.com/thetbw/claude-docker
cd claude-docker
# 2. еңЁйЎ№зӣ®дёӢж·»еҠ дёҖдёӘ.envж–Үд»¶е№¶зј–иҫ‘пјҢжҜ”еҰӮдёҠйқўзҡ„иҮӘе®ҡд№үд»ЈзҗҶпјҢе°ұеҸҜд»ҘеҠ еңЁиҝҷйҮҢ
# дҪҝз”Ёд»ЈзҗҶзҡ„иҜқпјҢиҝҳйңҖиҰҒеңЁеҪ“еүҚйЎ№зӣ®зӣ®еҪ•дёӢж·»еҠ дёҖдёӘдёҠиҝ°зҡ„.claude.jsonжқҘи·іиҝҮеҲқе§ӢеҢ–
cp .env.example .env
nano .env # Add any optional configs
# 3. е®үиЈ…пјҢиҝҷйҮҢзҡ„е®үиЈ…ж”ҜжҢҒжҠҠи„ҡжң¬ж·»еҠ еҲ°дҪ зҡ„зҺҜеўғеҸҳйҮҸ
sudo ./src/install.sh
# 4. еңЁд»»дҪ•зӣ®еҪ•пјҢиҫ“е…Ҙclaude-dockerе°ұеҸҜд»Ҙжү§иЎҢдәҶгҖӮ
cd ~/your-project
claude-docker
```
![image.png](https://thetbw-hk.cos.thetbw.xyz/blog/image\_1768045477826.png)
жңҖеҗҺпјҢиҜ•дәҶдёӢ Mac дёҠзҡ„дёҖдёӘdockerз®ЎзҗҶе·Ҙе…· `OrbStack` иҝҳжҢәеҘҪз”Ёзҡ„пјҢжҜ”еҰӮ`Docker Desktop` зңӢзқҖиҲ’жңҚеӨҡдәҶгҖӮ
---
зҺ°еңЁйғҪиҜҙз”Ё AI жқҘйҷҚжң¬еўһж•ҲпјҢдҪҶеҜ№дәҺжү“е·ҘдәәжқҘиҜҙпјҢе°ұжҳҜжңүе°‘иөҡдәҶй’ұеҸҲеӨҡе№ІдәҶеҫҲеӨҡжҙ»гҖӮе°ұжӢҝеҲҡжүҚзҡ„ AI жқҘиҜҙпјҢиҰҒжҳҜз”Ё Claude е®ҳж–№зҡ„жқҘиҜҙпјҢдёҖдёӘжңҲеҮ зҷҫеқ—й’ұдәҶпјҢжҲ‘жІЎжіЁж„Ҹе…·дҪ“еӨҡе°‘й’ұпјҢиҝҳжІЎеҒҡд»Җд№ҲдәӢпјҢе…¬еҸёд№ҹеҫҲе°‘дјҡеё®дҪ жҠҘй”Җиҝҷдәӣй’ұпјҢдҪҶжҳҜжңүдәӣе…¬еҸёжҳҜжңүеҶ…йғЁзҡ„дёҖдәӣйҮҮиҙӯеҗ§гҖӮ
д»ҺдёҠж¬ЎиҜҙиҜ•иҜ•дёҖдәӣAIжқҘзңӢпјҢжҲ‘иҝҳжІЎжңүж·ұеәҰдҪҝз”Ёд»Җд№ҲпјҢд№ӢеүҚз”ЁиҝҮеҮ ж¬Ў`Roo Code`пјҢеҶҷдёҖдёӘе°ҸеҠҹиғҪе°ұиҰҒеҮ еқ—й’ұпјҢдёҖйЎҝжҠҳи…ҫд»ҠеӨ©йҘӯй’ұе°ұжІЎдәҶпјҢиҖҢдё”зңҹзҡ„дёҖиЁҖйҡҫе°ҪпјҢиҮӘе·ұиҝҳиҰҒжңүжӣҙеӨҡзҡ„еҝғжҷәиҙҹжӢ…пјҢеҺҹдёҚеҰӮзҶҹз»ғжҺҢжҸЎдёҖдёӘжҠҖиғҪжңүз”ЁпјҢжүҖд»Ҙж„ҹи§үпјҢиҝҳжҳҜеҫ—иҮӘе·ұеӨҡеӯҰд№ е‘ҖгҖӮ
еҗҺйқўйЎәзқҖиҝҷж¬ЎзҒ«еұұжЁЎеһӢзҡ„еҢ…жңҲпјҢеҶҚиҜ•дёҖдёӘжңҲзңӢзңӢеҗ§пјҢиҝҷеӨ©жҲ‘жү“еӯ—йғҪж„ҹи§үеҶ»жүӢ...
