## Wine 11.0 释出

[![Wine](https://icon.solidot.org/images/topics/topicWine.png?123)](/search?tid=126 "Wine")

[Edwards](/~Edwards) (42866)发表于 2026年01月14日 14时38分 星期三 [新浪微博分享](//service.weibo.com/share/share.php?url=//www.solidot.org/story?sid=83299&appkey=1370085986&title=Wine%2011.0%20%E9%87%8A%E5%87%BA "新浪微博分享")
![](https://icon.solidot.org/images/a7c7.png)

**来自棕榈上的霜**

Wine 11.0 正式释出。主要变化包括：全新的 WoW64 模式，支持 32 位甚至 16 位应用在 64 位前缀下以更简洁的方式运行；支持 Linux 6.14 的 NTSYNC 内核模块，显著改进游戏和多线程应用性能；统一的 Wine 二进制文件；支持 Vulkan API 1.4.335；通过 Direct3D 实现硬件加速的 H.264 解码；改进 Wine Wayland 驱动，支持剪贴板操作、IME 和创建非矩形窗口；等等。
https://gitlab.winehq.org/wine/wine/-/releases/wine-11.0
