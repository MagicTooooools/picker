火电厂组成

![](https://mmbiz.qpic.cn/mmbiz_png/icKb2wsWzy9OuiaUiagrAFkgLtAvFjTq4Z4nf75PyoD68Vq1sorfNV88qzfYNECbVPZgcKrU2ueicIpKyPGjH10ggg/640?wx_fmt=png&from=appmsg)

它由下列5个系统组成

①燃料系统。

②燃烧系统（锅炉系统）。

③汽水系统。

④电气系统。

⑤控制系统。

在上述系统中，最主要的设备是锅炉、汽轮机和发电机，它们安装在发电厂的主厂房内。

主变压器和配电装置一般装放在独立的建筑物内或户外，这种设备属于涉网部分，在发电厂涉网部分还有AVC、AGC等系统。

其他辅助设备如给水系统、供水设备、水处理设备、除尘设备、燃料储运设备等，有的安装在主厂房内，有的则安装在辅助建筑中或在露天场地。

火电厂基本生产过程是，燃料在锅炉中燃烧，将其热量释放出来，传给锅炉中的水，从而产生高温高压蒸汽；蒸汽通过汽轮机又将热能转化为旋转动力，以驱动发电机输出电能。

火力发电厂控制系统组成：

①主控：DCS，包括锅炉、汽轮机、发电机、脱硫脱硝系统、炉膛安全监控系统FSSS等

②辅控：PLC，水处理、输煤、输灰等

三大系统

•燃烧系统：由输煤、制粉、锅炉与燃烧、风烟系统、除尘、脱硫等组成。

•汽水系统：由锅炉、汽轮机、凝汽器、高低压加热器、凝结水泵、给水泵等组成，主要包括汽水循环、化学水处理和冷却系统等。

•电气系统：汽轮发电机控制系统、厂用电控制系统、直接送出线路或升压变电站控制系统。

现代大型火电厂的这三大系统控制，已高度集成在分散控制系统（DCS） 中，实现“机-炉-电”一体化协调控制。

燃烧系统控制

燃烧系统的控制目标是：根据电网负荷要求，**精确、安全地组织燃料燃烧，并确保烟气排放达标**。其核心是维持**燃料、风量、负荷**三者间的动态平衡。

#### **1. 核心控制回路：锅炉协调控制**

* **作用原理**：这是整个机组自动控制的“大脑”。它接收来自电网的**负荷指令**，经过运算后，同时向锅炉主控和汽轮机主控发出指令。

+ **锅炉主控**：通过调节**给煤量**和**总风量**，改变锅炉的“产热”能力。
+ **汽轮机主控**：通过调节**汽轮机调门开度**，改变蒸汽流量和“用热”能力。
+ **原理**：采用“机跟炉”、“炉跟机”或“机炉协调”模式。在“协调模式”下，当负荷变化时，锅炉和汽轮机同时动作，既快速响应负荷，又维持主蒸汽压力稳定，实现动态平衡。

#### **2. 燃料子系统控制**

* **给煤机控制**：接收锅炉主控的燃料量指令，精确调节给煤机转速（皮带秤原理），控制进入磨煤机的原煤量。
* **磨煤机控制**：

+ **风量/煤量比**：保证磨煤机出口风粉混合物具有合适的温度和流速，既满足燃烧要求，又防止煤粉堵塞或自燃。
+ **出口温度控制**：通过调节冷、热风门的开度，将温度控制在安全范围内（通常~70℃），防止温度过高引起爆燃。

* **一次风控制**：为煤粉输送提供动力，其压力控制必须稳定，以确保各燃烧器煤粉分配的均匀性。

#### **3. 风烟子系统控制（燃烧优化核心）**

* **送风控制（氧量控制回路）**：

+ **原理**：送风量的目标是提供适量的氧气使燃料充分燃烧。控制系统实时监测**烟气含氧量**，并将其与设定值（根据负荷和煤种优化）比较。
+ **闭环调节**：若氧量偏低，说明风量不足，会导致燃烧不完全，增加煤耗并产生一氧化碳；若氧量偏高，则排烟热损失增大。控制系统通过调节送风机入口导叶或动叶，动态调整总风量，使氧量维持在最优值。

* **引风控制（炉膛负压控制回路）**：

+ **原理**：炉膛必须维持**微负压**（如-50 Pa）。负压过小或正压会导致火焰和烟气外喷，危及安全；负压过大则漏风严重，降低效率。
+ **闭环调节**：实时监测炉膛压力，通过调节引风机出力，使其快速跟随送风量的变化，维持压力恒定。这是一个快速、灵敏的单回路控制。

#### **4. 环保子系统控制**

* **脱硝（SCR/SNCR）控制**：

+ **原理**：核心是控制**喷氨量**。系统连续监测**烟气出口NOx浓度**和烟气流量，通过前馈-反馈复合算法，计算出精确的氨需求量，并控制喷氨阀开度。
+ **目标**：在保证NOx达标排放的前提下，最小化氨逃逸（过量氨），防止空预器堵塞和腐蚀。

* **脱硫（FGD）控制**：

+ **原理**：核心是控制**吸收塔浆液pH值**和**浆液密度**。
+ **pH值控制**：通过调节石灰石浆液的供给量来维持。pH值影响SO₂的吸收效率。
+ **密度控制**：通过定时排出石膏来控制，保证浆液品质和系统稳定运行。

#### **5. 安全守护神：炉膛安全监控系统（FSSS）**

* **作用原理**：它不负责连续调节，而是一套基于**硬逻辑**的**顺序控制和保护系统**（常采用独立PLC并与DCS通讯）。
* **功能**：

+ **点火前炉膛吹扫**：强制执行“所有燃料切断，大风量吹扫至少5分钟”的逻辑，清除炉内可能积聚的可燃物。
+ **点火许可和顺序启停**：严格检查各项条件（如油压、火焰检测器正常、风量合适），条件全部满足才允许点火。
+ **全炉膛灭火保护（MFT）**：这是最高级别的保护。当检测到“所有燃料丧失”、“炉膛压力极高/极低”、“全炉膛灭火”等任一危险信号时，FSSS会**毫秒级动作**，切断所有进入炉膛的燃料（停给煤机、关燃油阀），并引发相关设备联锁动作，防止爆炸。

### **二、 汽水系统控制：追求平稳、高效、安全的“水与汽”循环**

汽水系统的控制目标是：在锅炉侧**安全地产生合格蒸汽**，在汽轮机侧**高效地利用蒸汽做功**，并**回收工质、维持循环**。

#### **1. 锅炉侧核心控制**

* **汽包水位控制（三冲量控制）**：

+ **原理**：锅炉最重要的控制之一。水位过高影响蒸汽品质，过低则危及锅炉安全。
+ **三冲量**指：**汽包水位（主调量）、蒸汽流量（前馈量）、给水流量（反馈量）**。
+ **作用**：当负荷（蒸汽流量）变化时，控制器能提前根据蒸汽流量的变化增减给水量（前馈作用），再根据水位偏差和给水流量反馈进行精细修正，有效克服“虚假水位”现象，控制极为平稳。

* **主蒸汽温度控制**：

+ **原理**：蒸汽温度是影响机组效率和汽轮机安全的关键参数。通常采用**串级控制**。
+ **主调**：以过热器出口汽温为最终目标。
+ **副调**：以喷水减温器后的汽温或喷水流量为中间参数，快速响应扰动，调节减温水调节阀开度。通常布置多级减温，分段控制。

* **主蒸汽压力控制**：如前所述，主要通过锅炉协调控制系统中的燃料量调节来保证。

#### **2. 汽轮机侧核心控制**

* **数字电液调节系统（DEH）**：

+ **作用原理**：这是汽轮机的“专用大脑”。它接收协调控制系统或操作员的转速/负荷指令。
+ **转速控制**：在启动阶段，通过控制**高压调门**开度，精确控制汽轮机升速、暖机、过临界转速，直至达到3000r/min额定转速。
+ **负荷控制**：并网后，通过控制调门开度来调节进汽量，从而改变机组出力。它能实现一次调频功能（当电网频率变化时，自动微调负荷以辅助稳定电网）。

* **汽轮机危急遮断系统（ETS）**：

+ **作用原理**：汽轮机的“紧急刹车”。它独立于DEH，监视**超速（110%）、轴向位移大、轴承振动高、润滑油压低、EH油压低**等关键安全参数。任一参数超标，ETS会立即动作，使**安全油泄压**，触发机械机构快速关闭所有主汽门和调门，强制停机。

#### **3. 辅机侧核心控制**

* **除氧器压力和水位控制**：压力控制保证除氧效果；水位控制保证给水泵安全运行。
* **凝汽器热井水位控制**：调节凝结水泵出力或再循环门，维持水位稳定，保证真空度和凝结水泵不汽蚀。
* **加热器水位控制**：调节疏水阀，维持水位在正常范围，保证传热效果并防止汽轮机进水。

### **电气系统控制：追求可靠、稳定、同步的“电”**

电气系统的控制目标是：**安全地发出符合电网质量要求的电能，并保障厂用电不间断供应**。

#### **1. 发电机-变压器组控制**

* **自动电压控制（AVC）**：

+ **作用原理**：控制发电机的**无功功率**和出口电压。
+ **闭环调节**：AVC装置接收来自电网调度或设定的母线电压目标值，与实测电压比较，计算出所需的无功出力，然后自动向发电机的**励磁调节器（AVR）** 发出增、减磁指令，调整转子励磁电流，从而改变发电机端电压和无功输出，维持电网电压稳定。

* **自动发电控制（AGC）**：

+ **作用原理**：控制发电机的**有功功率**。
+ **闭环调节**：AGC接收来自电网调度中心的**全厂负荷指令**，根据机组的经济性能和响应能力，将指令分配给各台机组。机组协调控制系统则根据此指令，调整汽轮机的DEH和锅炉燃烧，改变实发有功功率，以满足电网的总负荷需求，参与电网的二次调频。

#### **2. 厂用电系统控制**

* **作用原理**：确保给锅炉、汽轮机等辅机供电的6kV/400V系统绝对可靠。

+ **备用电源自动投入装置（BZT）**：实时监测工作电源的电压和电流。当检测到工作电源故障（如厂用变压器跳闸）时，BZT会在几十毫秒内自动合上备用电源开关，保证关键辅机（如给水泵、送引风机）不停机，这是机组安全停运甚至快速恢复的基础。
+ **厂用电快切装置**：更为先进的快速切换逻辑，确保在机组启停或故障时，厂用电母线相位、幅值在最佳时机切换，减少对电动机的冲击。

#### **3. 升压站及继电保护**

* **作用原理**：这部分控制（断路器分合闸）通常由独立的**网络监控系统（NCS）** 或综合自动化系统完成，并与DCS通信。

+ **顺序控制**：可实现线路或开关的程序化倒闸操作。
+ **同期并列**：机组并网时，通过**自动同期装置**精确调节发电机电压、频率和相位，使其与电网系统严格同步，满足条件后自动发出合闸命令，实现平滑无冲击并网。
+ **继电保护**：是电气系统的“FSSS/ETS”。当线路、变压器、发电机发生短路、接地等故障时，保护装置（如差动保护、距离保护）会毫秒级动作，跳开相应断路器，隔离故障设备，保护主设备安全。

**NCS 与DCS的数据交换**

| 数据类型 | 方向 (NCS ↔ DCS) | 具体内容举例 | DCS侧用途 |

1. **AGC**：自动发电控制，是电网调度中心为了维持电网频率和功率平衡，向电厂发出的负荷分配指令。电厂接收到AGC指令后，需要自动调整机组出力（有功功率）以满足指令要求。
2. **AVC**：自动电压控制，是电网调度中心为了维持电网电压稳定，向电厂发出的电压或无功功率调整指令。电厂接收到AVC指令后，需要自动调整发电机励磁（无功功率）以满足指令要求。

### AGC的实现流程

在电厂中，AGC指令的接收和执行通常涉及以下步骤：

1. **指令接收**：电网调度中心通过电力数据网将AGC指令下发到电厂的**远动终端（RTU）**或**调度数据网**，进而到达电厂的**NCS**或专用的**AGC执行装置**。
2. **指令传递**：NCS将接收到的AGC指令（通常是有功功率目标值）通过通讯接口装置发送给DCS。注意，这里的数据流是：调度中心 -> NCS -> 通讯接口装置 -> DCS。
3. **指令处理**：DCS的协调控制系统（CCS）接收到AGC指令后，将其与当前机组实际出力进行比较，计算出偏差。然后通过锅炉主控和汽轮机主控的协调，调整燃料量、风量、给水量和汽轮机调门，改变机组出力，使实际功率逼近AGC指令。
4. **反馈**：机组实际出力的信号通过DCS采集，再通过NCS上送给调度中心，形成闭环控制。

所以，AGC的联动路径是：调度中心 <-> NCS <-> DCS（CCS）<-> 机组。

### 二、AVC的实现流程

AVC的实现与AGC类似，但控制对象是无功功率或电压：

1. **指令接收**：调度中心将AVC指令（可以是母线电压目标值或无功功率目标值）下发到电厂的NCS或专用的AVC装置。
2. **指令传递**：NCS将AVC指令发送给DCS。注意，有些电厂AVC控制是在专用的AVC装置中完成的，但需要与DCS进行数据交换。
3. **指令处理**：DCS将AVC指令传递给发电机的励磁调节系统（AVR）。励磁系统根据指令调整发电机励磁电流，从而改变发电机输出的无功功率和机端电压。
4. **反馈**：调整后的电压或无功功率通过NCS上送给调度中心。

所以，AVC的联动路径是：调度中心 <-> NCS <-> DCS（或直接到励磁系统）<-> 发电机励磁系统。

### 三、数据流和控制的自动化

整个过程中，NCS和DCS之间的数据交换是自动的，不需要人工干预。具体来说：

1. **数据交换点**：

* NCS将调度下发的AGC/AVC指令以模拟量或整数值的形式发送给DCS。
* DCS将机组的实际有功、无功、电压等信号发送给NCS，由NCS上送调度。

2. **控制自动化**：

* 在DCS侧，协调控制系统（CCS）和自动电压控制逻辑（可能集成在DCS中，也可能是独立装置）会实时接收AGC/AVC指令，并自动调整控制输出，使机组跟踪指令。

3. **安全措施**：

* 在指令传输过程中，会有各种限制和闭锁逻辑。例如，当机组处于手动模式或故障时，会自动退出AGC/AVC。同时，指令的变化率和幅度也会受到限制，确保机组安全。

4. **联动示例**：

* 当调度需要增加电厂出力时，下发更高的AGC有功指令。NCS接收后传给DCS，DCS的CCS增加燃料量、风量，开大汽轮机调门，使发电机出力增加。同时，由于出力增加，机端电压可能变化，AVC也会自动调整励磁以维持电压在要求范围。
